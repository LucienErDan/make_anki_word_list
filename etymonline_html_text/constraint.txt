<p>This is apparently from a root <span class="foreign">*nauti-</span> &quot;death, to be exhausted,&quot; source also of Old English <span class="foreign">ne, neo</span>, Old Norse <span class="foreign">na</span>, Gothic <span class="foreign">naus</span> &quot;corpse;&quot; Old Irish <span class="foreign">naunae</span> &quot;famine, shortage,&quot; Old Cornish <span class="foreign">naun</span> &quot;corpse;&quot; Old Church Slavonic <span class="foreign">navi</span> &quot;corpse,&quot; <span class="foreign">nazda</span>, Russian <span class="foreign">nuzda</span>, Polish <span class="foreign">nędza</span> &quot;misery, distress;&quot; Old Prussian <span class="foreign">nowis</span> &quot;corpse,&quot; <span class="foreign">nautin</span> &quot;need, distress,&quot; <span class="foreign">nawe</span> &quot;death;&quot; Lithuanian <span class="foreign">novyti</span> &quot;to torture, kill,&quot; <span class="foreign">nove</span> &quot;death.&quot; As it is attested only in Germanic, Celtic, and Balto-Slavic, it might be non-PIE, from a regional substrate language.</p>
