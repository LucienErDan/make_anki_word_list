                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/anathema">anathema</a> 诅咒；革出教门；被诅咒者</span><p>希腊前缀ana-这里表up，与on同源，词根the-表to put，后缀-ma表行为对象和结果。anathema本指将掠夺来的敌人的物品置于寺庙（或祭坛）之上来诅咒敌人厄运。历史上最出名的anathema是1054年导致基督教大分裂的东、西罗马帝国教会的相互开除教籍。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/anathema">anathema</a></dt><dd><b>anathema: </b><span>[16] Originally in Greek <i>anáthēma </i>was a ‘votive offering’ (it was a derivative of the compound verb <i>anatithénai </i>‘set up, dedicate’, formed from the prefix <i>ana</i>- ‘up’ and the verb <i>tithénai </i>‘place’, source of English <i>theme </i>and related to English <i>do</i>). But from being broadly ‘anything offered up for religious purposes’, the word gradually developed negative associations of ‘something dedicated to evil’; and by the time it reached Latin it meant ‘curse’ or ‘accursed person’.</span><br><span class="cognate">=&gt; <a href="/etym/s/do" target="_blank">do</a>, <a href="/etym/s/theme" target="_blank">theme</a></span></dd><dt><a target="_blank" href="/etym/s/anathema">anathema (n.)</a></dt><dd>1520s, "an accursed thing," from Latin <span class="foreign">anathema</span> "an excommunicated person; the curse of excommunication," from Greek <span class="foreign">anathema</span> "a thing accursed," originally "a thing devoted," literally "a thing set up (to the gods)," from <span class="foreign">ana-</span> "up" (see <a href="/etym/s/ana-" target="_blank">ana-</a>) + <span class="foreign">tithenai</span> "to put, place" (see <a href="/etym/s/theme" target="_blank">theme</a>).&#13; <br /><br />&#13; Originally simply a votive offering, by the time it reached Latin the meaning had progressed through "thing devoted to evil," to "thing accursed or damned." Later applied to persons and the Divine Curse. Meaning "formal act or formula of consigning to damnation" is from 1610s.&#13; <br /><br /><span class="foreign">Anathema maranatha</span>, taken as an intensified form, is a misreading of the Syriac <span class="foreign">maran etha</span> "the Lord hath come," which follows <span class="foreign">anathema</span> in I Cor. xvi:22, but is not connected with it (see <a href="/etym/s/Maranatha" target="_blank">Maranatha</a>).</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
