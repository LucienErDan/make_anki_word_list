                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/scabbard">scabbard</a> 剑鞘</span><p>来自盎格鲁法语*escauberc,剑鞘，来自 Proto-Germanic*sker-berg,剑鞘，来自 PIE*sker,砍，劈， 词源同 score,shear,-berg,保护，来自 PIE*berg,保护，词源同 burg,bury.词义由砍，劈引申为刀， 剑。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/scabbard">scabbard</a></dt><dd><b>scabbard: </b><span>[13] English acquired <i>scabbard </i>from Anglo-Norman <i>escaubers</i>. This appears to have been a compound formed from Old High German <i>scār</i>, which usually meant ‘scissors’ but was also used for ‘sword’ (it came from the same base that produced English <i>shear</i>), and the element -<i>berc </i>‘protection’ (as in <i>hauberk </i>[13], which etymologically means ‘neck-protection’), which was derived from <i>bergan </i>‘protect’ (a relative of English <i>borough</i>, <i>borrow</i>, <i>bury</i>, etc). So essentially, a <i>scabbard </i>is ‘sword-protection’.</span><br><span class="cognate">=&gt; <a href="/etym/s/borough" target="_blank">borough</a>, <a href="/etym/s/borrow" target="_blank">borrow</a>, <a href="/etym/s/bury" target="_blank">bury</a>, <a href="/etym/s/share" target="_blank">share</a>, <a href="/etym/s/shear" target="_blank">shear</a>, <a href="/etym/s/shirt" target="_blank">shirt</a>, <a href="/etym/s/short" target="_blank">short</a>, <a href="/etym/s/skirt" target="_blank">skirt</a></span></dd><dt><a target="_blank" href="/etym/s/scabbard">scabbard (n.)</a></dt><dd>c. 1300, from Anglo-French <span class="foreign">*escauberc</span> "sheath, vagina" (13c.), from Frankish or another Germanic source, from Proto-Germanic <span class="foreign">*sker-berg-</span>, literally "sword-protector," from <span class="foreign">*skar</span> "blade" (source also of Old High German <span class="foreign">scar</span> "scissors, blade, sword," from PIE <span class="foreign">*(s)ker-</span> (1) "to cut;" see <a href="/etym/s/shear" target="_blank">shear</a> (v.)) + <span class="foreign">*berg-</span> "protect" (source also of Old High German <span class="foreign">bergan</span> "to protect;" see <a href="/etym/s/bury" target="_blank">bury</a>).</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
