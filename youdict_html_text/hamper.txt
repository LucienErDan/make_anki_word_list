                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/hamper">hamper</a> 带盖大篮子，妨碍，阻止</span><p>来自古法语hanap,杯子，酒杯，后引申词义装杯子的大篮子。防碍义词源不详，参照习语basket case,困境。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/hamper">hamper (v.)</a></dt><dd>late 14c., <span class="foreign">hampren</span> "to surround, imprison, confine," also "to pack in a container;" of uncertain origin; probably from <a href="/etym/s/hamper" target="_blank">hamper</a> (n.1), unless it is somehow connected to Middle English <span class="foreign">hamelian</span> "to maim." Meaning "impede in motion or progress" is from late 14c. Related: <span class="foreign">Hampered</span>; <span class="foreign">hampering</span>.</dd><dt><a target="_blank" href="/etym/s/hamper">hamper (n.1)</a></dt><dd>"large basket," early 14c., <span class="foreign">hampyre</span>, probably a contraction of Anglo-French <span class="foreign">hanaper</span> (Anglo-Latin <span class="foreign">hanepario</span>), from Old French <span class="foreign">hanepier</span> "case for holding a large goblet or cup;" in medical use "skull," also "helmet; armored leather cap," from <span class="foreign">hanap</span> "goblet, chalice," from Frankish or some other Germanic source (cognates: Old Saxon <span class="foreign">hnapp</span> "cup, bowl;" Old High German <span class="foreign">hnapf</span>, German <span class="foreign">Napf</span>, Old English <span class="foreign">hnæpp</span>). The first <span class="foreign">-a-</span> may be a French attempt to render Germanic <span class="foreign">hn-</span> into an acceptable Romanic form. The English word also meant "the department of Chancery into which fees were paid for sealing and enrolling charters, etc." (15c.).</dd><dt><a target="_blank" href="/etym/s/hamper">hamper (n.2)</a></dt><dd>"things important for a ship but in the way at certain times" (Klein's definition), 1835, from <span class="foreign">hamper</span> (n.) "a fetter, shackles," from French <span class="foreign">hamper</span> "to impede." Hence <span class="foreign">top hamper</span>, originally "upper masts, spars, rigging, etc. of a sailing ship."</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
