            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/ambidextrous">ambidextrous</a> 左右手都灵活的</span><p>前缀ambi-, 两边。词根dext, 右手，指灵活。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/ambidextrous">ambidextrous</a></dt><dd><b>ambidextrous: </b><span>[16] <i>Ambidextrous </i>means literally ‘right-handed on both sides’. It was formed in Latin from the prefix <i>ambi</i>- ‘both’ and the adjective <i>dexter </i>‘right-handed’ (source of English <i>dextrous</i>). <i>Ambi</i>- corresponds to the Latin adjective <i>ambo </i>‘both’, which derived ultimately from the Indo-European base *<i>amb</i>- ‘around’ (an element in the source of <i>ambassador </i>and <i>embassy</i>).<br><br>The second element in Latin <i>ambo </i>seems to correspond to Old English <i>ba </i>‘both’, which is related to modern English <i>both</i>. Other English words formed with the prefix <i>amb(i)</i>- include <i>ambient </i>[16] (which came, like <i>ambition</i>, from Latin <i>ambīre </i>‘go round’), <i>ambit </i>[16] (from Latin <i>ambitus </i>‘circuit’), <i>ambiguous, ambition, amble</i>, and <i>ambulance</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/dextrous" target="_blank">dextrous</a></span></dd><dt><a target="_blank" href="/etym/s/ambidextrous">ambidextrous (adj.)</a></dt><dd>1640s, with <a href="/etym/s/-ous" target="_blank">-ous</a>, from <span class="foreign">ambidexter</span> (adj.) "double-dealing" (1610s), from French <span class="foreign">ambidextre</span> or directly from Latin <span class="foreign">ambidexter</span>, literally "right-handed on both sides," from <span class="foreign">ambi</span>- "both" (see <a href="/etym/s/ambi-" target="_blank">ambi-</a>) + <span class="foreign">dexter</span> "right-handed" (see <a href="/etym/s/dexterity" target="_blank">dexterity</a>). Its opposite, <span class="foreign">ambilevous</span> "left-handed on both sides, clumsy" (1640s) is rare. <span class="foreign">Ambidexter</span> as a noun, "one who takes bribes from both sides," is attested from 1530s and is the earliest form of the word in English; its sense of "one who uses both hands equally well" appears by 1590s.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
