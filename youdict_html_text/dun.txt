            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/dun">dun</a> 灰褐色的</span><p>来自PIE*dheu,尘土，烟尘，词源同down, dust. 即尘土色的，灰褐色的。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/dun">dun</a></dt><dd><b>dun: </b><span>English has two words <i>dun</i>. The colour adjective, ‘greyish brown’ [OE], comes ultimately from Indo-European *<i>donnos</i>, *<i>dusnos</i>, which is also the source of English <i>dusk</i>. The now rather dated noun, ‘debtcollector’ [17], is an abbreviation of <i>dunkirk</i>, a 17th-century term for a ‘privateer’, a privately owned vessel officially allowed to attack enemy shipping during wartime.<br><br>It was originally applied from such privateers that sailed from the port of Dunkirk, on the northern coast of France, to attack British ships, and its connotations of unwarranted piracy soon spread metaphorically to one who was constantly importuning for the repayment of his loan.</span><br><span class="cognate">=&gt; <a href="/etym/s/donkey" target="_blank">donkey</a>, <a href="/etym/s/dusk" target="_blank">dusk</a>, <a href="/etym/s/obfuscate" target="_blank">obfuscate</a></span></dd><dt><a target="_blank" href="/etym/s/dun">dun (v.)</a></dt><dd>"to insist on payment of debt," 1620s, perhaps related to <span class="foreign">dunnen</span> "to sound, resound, make a din" (c. 1200, dialectal variant of <a href="/etym/s/din" target="_blank">din</a>), or shortened from <span class="foreign">dunkirk</span> (c. 1600) "privateer," a private vessel licensed to attack enemy ships during wartime, from <span class="foreign">Dunkirk</span>, French port from which they sailed. The oldest theory traces it to a Joe <span class="foreign">Dun</span>, supposedly a London bailiff famous for catching defaulters. Related: <span class="foreign">Dunned</span>; <span class="foreign">dunning</span>. As a noun from 1620s.</dd><dt><a target="_blank" href="/etym/s/dun">dun (adj.)</a></dt><dd>Old English <span class="foreign">dunn</span> "dingy brown, dark-colored," perhaps from Celtic (compare Old Irish <span class="foreign">donn</span> "dark;" Gaelic <span class="foreign">donn</span> "brown, dark;" Welsh <span class="foreign">dwnn</span> "brownish"), from PIE <span class="foreign">*donnos</span>, <span class="foreign">*dusnos</span> "dark."</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
