                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/brain">brain</a> 脑</span><p>词源不详。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/brain">brain</a></dt><dd><b>brain: </b><span>[OE] Old English <i>brægen </i>came from a Germanic *<i>bragnam</i>. Its rather restricted distribution in modern Germanic languages (apart from English <i>brain </i>there is only Dutch and Frisian <i>brein</i>) suggests that in prehistoric times it may have been limited to the area of North Germany where the Low German dialects were spoken, but it may well have some connection with Greek <i>brekhmós </i>‘forehead’.</span><br></dd><dt><a target="_blank" href="/etym/s/brain">brain (v.)</a></dt><dd>"to dash the brains out," late 14c., from <a href="/etym/s/brain" target="_blank">brain</a> (n.). Related: <span class="foreign">Brained</span>; <span class="foreign">braining</span>.</dd><dt><a target="_blank" href="/etym/s/brain">brain (n.)</a></dt><dd>Old English <span class="foreign">brægen</span> "brain," from Proto-Germanic <span class="foreign">*bragnam</span> (cognates: Middle Low German <span class="foreign">bregen</span>, Old Frisian and Dutch <span class="foreign">brein</span>), from PIE root <span class="foreign">*mregh-m(n)o-</span> "skull, brain" (cognates: Greek <span class="foreign">brekhmos</span> "front part of the skull, top of the head"). But Liberman writes that <span class="foreign">brain</span> "has no established cognates outside West Germanic ..." and is not connected to the Greek word. More probably, he writes, its etymon is PIE <span class="foreign">*bhragno</span> "something broken."&#13; <br /><br />&#13; The custom of using the plural to refer to the substance (literal or figurative), as opposed to the organ, dates from 16c. Figurative sense of "intellectual power" is from late 14c.; meaning "a clever person" is first recorded 1914. <span class="foreign">Brain teaser</span> is from 1923. <span class="foreign">Brain stem</span> first recorded 1879, from German. <span class="foreign">Brain drain</span> is attested from 1963. An Old English word for "head" was <span class="foreign">brægnloca</span>, which might be translated as "brain locker." In Middle English, <span class="foreign">brainsick</span> (Old English <span class="foreign">brægenseoc</span>) meant "mad, addled."</dd></dl></div></div>

            <div class="row" id="yd-liju">
                 <dl><dt>1. The horrors he experienced are imprinted, perhaps indelibly, in his <b>brain</b>. </dt> <dd>他经历的恐怖事件在他脑海中刻下了深深的，或许是难以磨灭的印记。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. She has a first-class <b>brain</b> and is a damned good writer. </dt> <dd>她头脑相当灵光，是个非常棒的作家。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>3. Hallucination is common in patients who have suffered damage to the <b>brain</b>. </dt> <dd>脑部受损的病人常会产生幻觉。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>4. He was witty, amusing and gifted with a sharp business <b>brain</b>. </dt> <dd>他机智诙谐，具有敏锐的商业头脑。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>5.     document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                        if(this.readyState == 'loaded' || this.readyState == 'complete') {
                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

        if(cstt) {
            _trans(); if(handler) handler.innerHTML = '简体版';
        } else {

            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';

                    window.localStorage.setItem('stt', cstt);
