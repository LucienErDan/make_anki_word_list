                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/trivial">trivial</a> 琐事的，小事的</span><p>trivia,琐事，小事，-al,形容词后缀。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/trivial">trivial</a></dt><dd><b>trivial: </b><span>[15] Medieval educationists recognized seven liberal arts: the lower three, grammar, logic, and rhetoric, were known as the <i>trivium</i>, and the upper four, arithmetic, astronomy, geometry, and music, were known as the <i>quadrivium</i>. The notion of ‘less important subjects’ led in the 16th century to the use of the derived adjective <i>trivial </i>for ‘commonplace, of little importance’. Latin <i>trivium </i>itself was a compound noun formed from the prefix <i>tri</i>- ‘three’ and <i>via </i>‘way, road’, and originally meant ‘place where three roads meet’.</span><br><span class="cognate">=&gt; <a href="/etym/s/three" target="_blank">three</a>, <a href="/etym/s/via" target="_blank">via</a></span></dd><dt><a target="_blank" href="/etym/s/trivial">trivial (adj.)</a></dt><dd>"ordinary" (1580s); "insignificant, trifling" (1590s), from Latin <span class="foreign">trivialis</span> "common, commonplace, vulgar," literally "of or belonging to the crossroads," from <span class="foreign">trivium</span> "place where three roads meet," in transferred use, "an open place, a public place," from <span class="foreign">tri-</span> "three" (see <a href="/etym/s/three" target="_blank">three</a>) + <span class="foreign">via</span> "road" (see <a href="/etym/s/via" target="_blank">via</a>). The sense connection is "public," hence "common, commonplace."&#13; <br /><br />&#13; The earliest use of the word in English was early 15c., a separate borrowing in the academic sense "of the trivium" (the first three <a href="/etym/s/liberal_arts" target="_blank">liberal arts</a> -- grammar, rhetoric, and logic); from Medieval Latin use of <span class="foreign">trivialis</span> in the sense "of the first three liberal arts," from <span class="foreign">trivium</span>, neuter of the Latin adjective <span class="foreign">trivius</span> "of three roads, of the crossroads." Related: <span class="foreign">Trivially</span>. For sense evolution to "pertaining to useless information," see <a href="/etym/s/trivia" target="_blank">trivia</a>.</dd></dl></div></div>
                                    <div class="page-header"><h3><a href="/liju/w/trivial">权威例句</a></h3></div>
            <div class="row" id="yd-liju">
                 <dl><dt>1. This may sound <b>trivial</b>, but I assure you it is quite important! </dt> <dd>这听上去也许微不足道，但是，相信我，它十分重要！</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. My mind attempted to calm itself by fastening on this <b>trivial</b> detail. </dt> <dd>我尝试将注意力集中在这个小细节上以让自己平静下来。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>3. I know it sounds <b>trivial</b> , but I'm worried about it. </dt> <dd>我知道这事听起来微不足道，但我还是放心不下。</dd> <p>来自《权威词典》</p> </dl><dl><dt>4. There are a few <b>trivial</b> slips in this lesson. </dt> <dd>在这篇课文中有几个小错误.</dd> <p>来自《简明英汉词典》</p> </dl><dl><dt>5. She works herself up about the most <b>trivial</b> things. </dt> <dd>她为了最琐碎的事也会使自己激动起来.</dd> <p>来自《简明英汉词典》</p> </dl>                 <dl><p>[ <a href="/liju/w/trivial">trivial 造句</a> ]</p></dl>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

            _trans(); if(handler) handler.innerHTML = '简体版';
        } else {

            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';

                    window.localStorage.setItem('stt', cstt);
