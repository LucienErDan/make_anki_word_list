                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/deed">deed</a></dt><dd><b>deed: </b><span>[OE] Etymologically, a <i>deed </i>is ‘that which is done’. An ancient word, it can be traced back as far as a hypothetical *<i>dhētis</i>, a noun derived from *<i>dhē</i>-, *<i>dhō</i>- ‘place, put’, the Indo- European base from which <i>do </i>comes. This passed into Germanic as *<i>dǣdiz</i>, which produced German <i>tat</i>, Dutch <i>daad</i>, and Swedish <i>dåad </i>as well as English <i>deed</i>. The word’s application to a legal document is a 14th-century development.</span><br><span class="cognate">=&gt; <a href="/etym/s/do" target="_blank">do</a></span></dd><dt><a target="_blank" href="/etym/s/deed">deed (n.)</a></dt><dd>Old English <span class="foreign">dæd</span> "a doing, act, action, transaction, event," from Proto-Germanic <span class="foreign">*dædis</span> (cognates: Old Saxon <span class="foreign">dad</span>, Old Norse <span class="foreign">dað</span>, Old Frisian <span class="foreign">dede</span>, Middle Dutch <span class="foreign">daet</span>, Dutch <span class="foreign">daad</span>, Old High German <span class="foreign">tat</span>, German <span class="foreign">Tat</span> "deed," Gothic <span class="foreign">gadeþs</span> "a putting, placing"), from PIE <span class="foreign">*dhetis</span> (cognates: Lithuanian <span class="foreign">detis</span> "load, burden," Greek <span class="foreign">thesis</span> "a placing, setting"), from <span class="foreign">*dhe-</span> "place, put" (see <a href="/etym/s/do" target="_blank">do</a>). Sense of "written legal document" is early 14c. As a verb, 1806, American English Related: <span class="foreign">Deeded</span>; <span class="foreign">deeding</span>.</dd></dl></div></div>

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {

            return false;
