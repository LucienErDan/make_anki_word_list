                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/magnet">magnet</a> 磁石，磁铁</span><p>来自拉丁语magnetum,磁石，来自希腊语Magnes lithos,即来自Magnesia的石头，lithos,石头，词源同lithology.Magnesia为古希腊地名，来自当地人的族名Magnetes.因石头中发现天然磁石而得名。比较magnesia.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/magnet">magnet</a></dt><dd><b>magnet: </b><span>[15] Greek <i>Mágnēs líthos </i>meant ‘stone from Magnesia’ – Magnesia being a region of Thessaly, Greece where much metal was obtained. It had two specific applications: to ore with magnetic properties, and to stone with a metallic sheen. And it was the first of these that has come down to English via Latin <i>magnēta </i>as <i>magnet</i>. English <i>magnesia </i>[14] comes from the same source, but it is not clear how it came to be applied (in the 18th century) to ‘magnesium oxide’, for it originally denoted, in the rather vague terminology of the alchemists, a ‘constituent of the philosopher’s stone’.<br><br>In the 17th century it was used for ‘manganese’ (and <i>manganese </i>[17] itself comes via French from Italian <i>manganese</i>, an alteration of medieval Latin <i>magnēsia</i>). And when the term <i>magnesium </i>[19] was introduced (at the suggestion of the chemist Sir Humphry Davy), it too at first denoted ‘manganese’.</span><br><span class="cognate">=&gt; <a href="/etym/s/magnesium" target="_blank">magnesium</a>, <a href="/etym/s/manganese" target="_blank">manganese</a></span></dd><dt><a target="_blank" href="/etym/s/magnet">magnet (n.)</a></dt><dd>mid-15c. (earlier <span class="foreign">magnes</span>, late 14c.), from Old French <span class="foreign">magnete</span> "magnetite, magnet, lodestone," and directly from Latin <span class="foreign">magnetum</span> (nominative <span class="foreign">magnes</span>) "lodestone," from Greek <span class="foreign">ho Magnes lithos</span> "the Magnesian stone," from <span class="foreign">Magnesia</span>, region in Thessaly where magnetized ore was obtained. Figurative use from 1650s. It has spread from Latin to most Western European languages (German and Danish <span class="foreign">magnet</span>, Dutch <span class="foreign">magneet</span>, Italian, Spanish, Portuguese <span class="foreign">magnete</span>), but it was superseded in French by <span class="foreign">aimant</span>. Also see <a href="/etym/s/magnesia" target="_blank">magnesia</a>. <span class="foreign">Chick magnet</span> attested from 1989.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
