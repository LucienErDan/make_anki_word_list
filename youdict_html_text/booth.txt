            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/booth">booth</a></dt><dd><b>booth: </b><span>[12] In common with a wide range of other English words, including <i>bower </i>and the -<i>bour </i>of <i>neighbour</i>, <i>booth </i>comes ultimately from the Germanic base *<i>bū</i>- ‘dwell’. From this source came the East Norse verb <i>bóa </i>‘dwell’ (whose present participle produced English <i>bond </i>and the -<i>band </i>of <i>husband</i>); addition of the suffix -<i>th </i>produced the unrecorded noun <i>bóth</i>. ‘dwelling’, which came into Middle English as <i>bōth</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/be" target="_blank">be</a>, <a href="/etym/s/boor" target="_blank">boor</a>, <a href="/etym/s/bower" target="_blank">bower</a>, <a href="/etym/s/husband" target="_blank">husband</a>, <a href="/etym/s/neighbour" target="_blank">neighbour</a></span></dd><dt><a target="_blank" href="/etym/s/booth">booth (n.)</a></dt><dd>mid-12c., from Old Danish <span class="foreign">boþ</span> "temporary dwelling," from East Norse <span class="foreign">*boa</span> "to dwell," from Proto-Germanic <span class="foreign">*bowan-</span>, from PIE root <span class="foreign">*bheue-</span> "to be, exist, grow" (see <a href="/etym/s/be" target="_blank">be</a>). See also <a href="/etym/s/bound" target="_blank">bound</a> (adj.2). Compare German <span class="foreign">Bude</span> "booth, stall," Middle Dutch <span class="foreign">boode</span>, Lithuanian <span class="foreign">butas</span> "house," Old Irish <span class="foreign">both</span> "hut," Bohemian <span class="foreign">bouda</span>, Polish <span class="foreign">buda</span>, some probably borrowed from East Norse, some formed from the PIE root.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
