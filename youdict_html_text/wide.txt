                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/wide">wide</a> 宽的</span><p>来自PIE*weidh,分开，词源同widow,with,divide,即分开的，宽的，进一步来自*dwi,二，词源同di-,two,*dhe,放置，词源同do,cred-,心，信任。即两边放置，分开。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/wide">wide</a></dt><dd><b>wide: </b><span>[OE] <i>Wide </i>is a general Germanic word, with relatives in German <i>weit</i>, Dutch <i>wijd</i>, and Swedish and Danish <i>vid</i>. All are descended from prehistoric Germanic *<i>wīdaz</i>, which may go back ultimately to the Indo-European base *<i>wi</i>- ‘apart, away’ (source also of Sanskrit <i>vitarám </i>‘further’). <i>Width </i>was coined in the early 17th century, probably on the analogy of <i>breadth</i>.</span><br></dd><dt><a target="_blank" href="/etym/s/wide">wide (adj.)</a></dt><dd>Old English <span class="foreign">wid</span> "vast, broad, long," also used of time, from Proto-Germanic <span class="foreign">*widaz</span> (cognates: Old Saxon, Old Frisian <span class="foreign">wid</span>, Old Norse <span class="foreign">viðr</span>, Dutch <span class="foreign">wijd</span>, Old High German <span class="foreign">wit</span>, German <span class="foreign">weit</span>), perhaps from PIE <span class="foreign">*wi-ito-</span>, from root <span class="foreign">*wi-</span> "apart, away, in half."&#13; <br /><br />&#13; Meaning "distended, expanded, spread apart" is from c. 1500; sense of "embracing many subjects" is from 1530s; meaning "missing the intended target" is from 1580s. As a second element in compounds (such as <a href="/etym/s/nationwide" target="_blank">nationwide</a>, <a href="/etym/s/worldwide" target="_blank">worldwide</a>) meaning "extending through the whole of," is is from late Old English. As an adverb, Old English <span class="foreign">wide</span>. <span class="foreign">Wide open</span> "unguarded, exposed to attack" (1915) originally was in boxing, etc. <span class="foreign">Wide awake</span> (adj.) is first recorded 1818; figurative sense of "alert, knowing" is attested from 1833.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

            _trans(); if(handler) handler.innerHTML = '简体版';

            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';

                    window.localStorage.setItem('stt', cstt);
