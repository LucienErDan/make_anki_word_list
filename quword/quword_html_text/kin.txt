                        <div class="page-header"><h3>记忆方法</h3></div>
            <div class="row" style="font-family:SimSun,serif;">1. *gen-, *gon-, *gn- "produce, born" => eg: generate, genital, nature, gonorrhoea.<br />
2. gen- => kin "family".<br />
3. kin + -red "condition, rule" (同后缀的其他词：hatred) => kindred.<br />
4. With intrusive -d- probably for phonetic reasons (cf. thunder) but perhaps encouraged by kind.<br />
5. 同源词：kind, 还有许多其它的词。</div>

                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/kin">kin</a></dt><dd><b>kin: </b><span>[OE] <i>Kin </i>is the central English member of the Germanic branch of a vast family of words that trace their ancestry back to the prehistoric Indo- European base *<i>gen</i>-, *<i>gon</i>-, *<i>gn</i>-, denoting ‘produce’ (the Latin branch has given English <i>gender</i>, <i>general</i>, <i>generate</i>, <i>genital</i>, <i>nature</i>, etc, the Greek branch <i>gene</i>, <i>genetic</i>, <i>gonorrhoea</i>, etc).<br><br>Amongst the Germanic descendants of this base was *<i>kun</i>-, from which was derived the noun *<i>kunjam</i>, source of Swedish <i>kön </i>‘sex’ and English <i>kin </i>‘family’. <i>Kindred </i>[12] was formed from <i>kin </i>in early Middle English by adding the suffix -<i>red </i>‘condition’ (as in <i>hatred</i>). Also closely related are <i>kind </i>and <i>king</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/gender" target="_blank">gender</a>, <a href="/etym/s/gene" target="_blank">gene</a>, <a href="/etym/s/general" target="_blank">general</a>, <a href="/etym/s/generate" target="_blank">generate</a>, <a href="/etym/s/genital" target="_blank">genital</a>, <a href="/etym/s/kind" target="_blank">kind</a>, <a href="/etym/s/kindred" target="_blank">kindred</a>, <a href="/etym/s/king" target="_blank">king</a>, <a href="/etym/s/nature" target="_blank">nature</a></span></dd><dt><a target="_blank" href="/etym/s/kin">kin (n.)</a></dt><dd>c. 1200, from Old English <span class="foreign">cynn</span> "family; race; kind, sort, rank; nature; gender, sex," from Proto-Germanic <span class="foreign">*kunjam</span> "family" (cognates: Old Frisian <span class="foreign">kenn</span>, Old Saxon <span class="foreign">kunni</span>, Old Norse <span class="foreign">kyn</span>, Old High German <span class="foreign">chunni</span> "kin, race;" Danish and Swedish <span class="foreign">kön</span>, Middle Dutch, Dutch <span class="foreign">kunne</span> "sex, gender;" Gothic <span class="foreign">kuni</span> "family, race," Old Norse <span class="foreign">kundr</span> "son," German <span class="foreign">Kind</span> "child"), from PIE <span class="foreign">*gene-</span> "to produce" (see <a href="/etym/s/genus" target="_blank">genus</a>).</dd></dl></div></div>
