                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/prior">prior</a> 先前的，优先的，小隐修院院长</span><p>来自拉丁语prior,先前的，领先的，来自古拉丁语pri,在前，先于，来自PIE*per,向前，词源同ford,first.-or,比较级后缀。后用于小隐修院院长尊称。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/prior">prior</a></dt><dd><b>prior: </b><span><i>Prior </i>the ecclesiastical rank [11] and <i>prior </i>‘previous’ [18] are ultimately the same word. Both go back to Latin <i>prior </i>‘former, superior’, a comparative formation based on the Old Latin preposition <i>pri </i>‘before’. This came in postclassical times to be used as a noun meaning ‘superior officer, administrator’, and it was taken over as such into the terminology of the monastic foundations.</span><br></dd><dt><a target="_blank" href="/etym/s/prior">prior (adj.)</a></dt><dd>"earlier," 1714, from Latin <span class="foreign">prior</span> "former, previous, first;" figuratively "superior, better;" as a noun "forefather; superior rank;" comparative of Old Latin <span class="foreign">pri</span> "before," from PIE <span class="foreign">*prai-</span>, <span class="foreign">*prei-</span>, from root <span class="foreign">*per-</span> (1) "forward, through" (see <a href="/etym/s/per" target="_blank">per</a>).</dd><dt><a target="_blank" href="/etym/s/prior">prior (n.)</a></dt><dd>"superior officer of a religious house or order," late Old English, from Medieval Latin <span class="foreign">prior</span> "superior officer," noun use of Latin adjective meaning "former, superior" (see <a href="/etym/s/prior" target="_blank">prior</a> (adj.)). As short for <span class="foreign">prior arrest</span>, by 1990, American English.</dd></dl></div></div>

            <div class="row" id="yd-liju">
                 <dl><dt>1. <b>Prior</b> to his Japan trip, he went to New York. </dt> <dd>在去日本之前，他去了纽约。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. Just phone in your order three or more days <b>prior</b> to departure. </dt> <dd>在出发的三天之前打电话预订即可。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>3. For the <b>prior</b> year, they reported net income of $1.1 million. </dt> <dd>他们公布上一年度纯收入为110万美元。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>4. Although not essential, some <b>prior</b> knowledge of statistics is desirable. </dt> <dd>统计学的知识虽非必要，但最好是学过一点。</dd> <p>来自《权威词典》</p> </dl><dl><dt>5. Anglers are required to obtain <b>prior</b> authorization from the park keeper. </dt> <dd>垂钓者必须事先得到公园管理者的许可.</dd> <p>来自《简明英汉词典》</p> </dl>                 <dl><p>[ <a href="/liju/w/prior">prior 造句</a> ]</p></dl>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
