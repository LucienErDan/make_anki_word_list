                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/seek">seek</a> 寻找，寻求</span><p>来自古英语 secan,寻求，追寻，渴望，来自 Proto-Germanic*sokjan,寻求，来自 PIE*sag,寻求， 找出，词源同 sake,beseech.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/seek">seek</a></dt><dd><b>seek: </b><span>[OE] <i>Seek </i>has several Germanic relatives – German <i>suchen</i>, Swedish <i>söka</i>, Danish <i>søge</i>, etc – which point back to a prehistoric Germanic ancestor *<i>sōkjan</i>. The base from which this was derived, *<i>sōk</i>-, went back to an Indo-European *<i>sāg</i>-, which also produced (via Latin) English <i>presage </i>[14] and <i>sagacious </i>[17]. If Old English <i>sēcan </i>had developed in the ordinary way, it would have become modern English <i>seech</i>, not <i>seek</i>. For various reasons it did not, but we can see how it would have been in its derivative <i>beseech </i>[12].</span><br><span class="cognate">=&gt; <a href="/etym/s/beseech" target="_blank">beseech</a>, <a href="/etym/s/presage" target="_blank">presage</a>, <a href="/etym/s/sagacious" target="_blank">sagacious</a>, <a href="/etym/s/sake" target="_blank">sake</a></span></dd><dt><a target="_blank" href="/etym/s/seek">seek (v.)</a></dt><dd>Old English <span class="foreign">secan</span> "inquire, search for; pursue; long for, wish for, desire; look for, expect from," influenced by Old Norse <span class="foreign">soekja</span>, both from Proto-Germanic <span class="foreign">*sokjan</span> (cognates: Old Saxon <span class="foreign">sokian</span>, Old Frisian <span class="foreign">seka</span>, Middle Dutch <span class="foreign">soekan</span>, Old High German <span class="foreign">suohhan</span>, German <span class="foreign">suchen</span>, Gothic <span class="foreign">sokjan</span>), from PIE <span class="foreign">*sag-yo-</span>, from root <span class="foreign">*sag-</span> "to track down, seek out" (cognates: Latin <span class="foreign">sagire</span> "to perceive quickly or keenly," <span class="foreign">sagus</span> "presaging, predicting," Old Irish <span class="foreign">saigim</span> "seek"). The natural modern form of the Anglo-Saxon word as uninfluenced by Norse is in <a href="/etym/s/beseech" target="_blank">beseech</a>. Related: <a href="/etym/s/Sought" target="_blank">Sought</a>; <span class="foreign">seeking</span>.</dd></dl></div></div>
