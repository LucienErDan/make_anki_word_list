                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/mauve">mauve</a> 淡紫色的</span><p>来自古法语mauve,来自拉丁语malva,锦葵，词源同mallow.即锦葵色的，淡紫色的。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/mauve">mauve</a></dt><dd><b>mauve: </b><span>[19] Etymologically, <i>mauve </i>is the colour of the ‘mallow’ flower. The word was borrowed from French <i>mauve</i>, whose original meaning was ‘mallow’, and which was descended from Latin <i>malva </i>‘mallow’. English took over <i>malva </i>in the Old English period as <i>mealuwe</i>, which has become modern English <i>mallow</i>. And Greek <i>molókhē </i>‘mallow’, a probable relative of <i>malva</i>, is the ultimate source of English <i>malachite </i>[16].</span><br><span class="cognate">=&gt; <a href="/etym/s/malachite" target="_blank">malachite</a>, <a href="/etym/s/mallow" target="_blank">mallow</a></span></dd><dt><a target="_blank" href="/etym/s/mauve">mauve (n.)</a></dt><dd>purple dye, 1859, from French <span class="foreign">mauve</span>, from Old French <span class="foreign">mauve</span> "mallow" (13c.), from Latin <span class="foreign">malva</span> "mallow;" the dye so called from the color of the mallow plant. Related: <span class="foreign">Mauvish</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
