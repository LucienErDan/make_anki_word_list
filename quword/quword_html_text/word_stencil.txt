                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/stencil">stencil</a> 模板，模板印制的图案</span><p>来自中古英语 stencellen,装饰以明亮图案，来自中古法语 estencele,闪光片，火星，火花，来 自通俗拉丁语*stincilla,置换自 scintilla,火星，火花，词源同 scintilla,scintillate.引申诸相关词 义。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/stencil">stencil</a></dt><dd><b>stencil: </b><span>[14] <i>Stencil </i>was originally a verb, meaning ‘decorate with bright colours’. It came from Old French <i>estenceler </i>‘cause to sparkle’, a derivative of <i>estencele </i>‘spark’. This was descended from Vulgar Latin *<i>stincilla</i>, an alteration of Latin <i>scintilla </i>‘spark’ (source of English <i>scintilla </i>‘jot’ [17] and <i>scintillate </i>[17]). There are no records of this original verb beyond the 15th century, and the noun <i>stencil </i>‘sheet with cut-out designs’ did not appear until the early 18th century, but despite the long gap, they are generally assumed to be the same word.</span><br><span class="cognate">=&gt; <a href="/etym/s/scintillate" target="_blank">scintillate</a>, <a href="/etym/s/tinsel" target="_blank">tinsel</a></span></dd><dt><a target="_blank" href="/etym/s/stencil">stencil (n.)</a></dt><dd>1707, not recorded again until 1848, probably from Middle English <span class="foreign">stencellen</span> "decorate with bright colors," from Middle French <span class="foreign">estenceler</span> "cover with sparkles or stars, powder with color," from <span class="foreign">estencele</span> "spark, spangle" (Modern French <span class="foreign">étincelle</span>), from Vulgar Latin <span class="foreign">*stincilla</span>, metathesis of Latin <span class="foreign">scintilla</span> "spark" (see <a href="/etym/s/scintilla" target="_blank">scintilla</a>).</dd><dt><a target="_blank" href="/etym/s/stencil">stencil (v.)</a></dt><dd>"to produce a design with a stencil," 1861, from <a href="/etym/s/stencil" target="_blank">stencil</a> (n.). Related: <span class="foreign">Stenciled</span>; <span class="foreign">stenciling</span> (1781 as a verbal noun).</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
