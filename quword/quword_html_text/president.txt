            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/president">president</a> 主席，总统，总裁</span><p>来自preside,主持，担任主席，-ent,人。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/president">president</a></dt><dd><b>president: </b><span>[14] A <i>president </i>is etymologically simply someone who ‘presides’. The word comes via Old French <i>president </i>from Latin <i>praesidēns</i>, the present participle of <i>praesidēre </i>‘superintend’ (it literally meant ‘sit in front of’ – it was formed from the prefix <i>prae</i>- ‘before’ and <i>sedēre </i>‘sit’ – and it has given English <i>preside </i>[17]). Another Latin derivative was the noun <i>praesidium </i>‘garrison, fortification’, which English has acquired via Russian <i>prezídium </i>as <i>presidium </i>[20].</span><br><span class="cognate">=&gt; <a href="/etym/s/preside" target="_blank">preside</a>, <a href="/etym/s/presidium" target="_blank">presidium</a>, <a href="/etym/s/sit" target="_blank">sit</a></span></dd><dt><a target="_blank" href="/etym/s/president">president (n.)</a></dt><dd>late 14c., "appointed governor of a province; chosen leader of a body of persons," from Old French <span class="foreign">president</span> and directly from Latin <span class="foreign">praesidentum</span> (nominative <span class="foreign">praesidens</span>) "president, governor," noun use of present participle of <span class="foreign">praesidere</span> "to act as head or chief" (see <a href="/etym/s/preside" target="_blank">preside</a>).&#13; <br /><br />&#13; In Middle English of heads of religious houses, hospitals, colleges and universities. First use for "chief executive officer of a republic" is in U.S. Constitution (1787), from earlier American use for "officer in charge of the Continental Congress" (1774), a sense derived from that of "chosen head of a meeting or group of persons," which is from Middle English. It had been used of chief officers of banks from 1781, of individual colonies since 1608 (originally Virginia) and heads of colleges since mid-15c. Slang shortening <span class="foreign">prez</span> is recorded from 1883. Fem. form <span class="foreign">presidentess</span> is attested from 1763.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                }

                _trans(true);handler.innerHTML = '繁體版';
