                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/search">search</a> 搜索，搜寻</span><p>来自古法语 cerchier,搜查，搜索，来自拉丁语 circare,搜查，绕圈，来自 circus,圈，环，词源 同 circus,recherche.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/search">search</a></dt><dd><b>search: </b><span>[14] Etymologically, <i>search </i>denotes ‘going round in a circle’ – for its ultimate source is Latin <i>circus </i>‘circle’ (source of English <i>circle </i>and <i>circus</i>). From this was derived the verb <i>circāre </i>‘go round’, which by the time it had reached Old French as <i>cerchier </i>had acquired connotations of ‘examining’ or ‘exploring’. English took it over via Anglo-Norman <i>sercher</i>. (It is no relation, incidentally, to English <i>seek</i>.)</span><br><span class="cognate">=&gt; <a href="/etym/s/circle" target="_blank">circle</a>, <a href="/etym/s/circus" target="_blank">circus</a></span></dd><dt><a target="_blank" href="/etym/s/search">search (v.)</a></dt><dd>c. 1300, from Old French <span class="foreign">cerchier</span> "to search" (12c., Modern French <span class="foreign">chercher</span>), from Latin <span class="foreign">circare</span> "go about, wander, traverse," in Late Latin "to wander hither and thither," from <span class="foreign">circus</span> "circle" (see <a href="/etym/s/circus" target="_blank">circus</a>). Phrase <span class="foreign">search me</span> as a verbal shrug of ignorance first recorded 1901. <span class="foreign">Search engine</span> attested from 1988. <span class="foreign">Search and destroy</span> as a modifier is 1966, American English, from the Vietnam War. <span class="foreign">Search and rescue</span> is from 1944.</dd><dt><a target="_blank" href="/etym/s/search">search (n.)</a></dt><dd>c. 1400, "act of searching;" early 15c., "right to investigate illegal activity; examination of records, wills, etc.; a search through an area or a place," from Anglo-French <span class="foreign">serche</span>, Old French <span class="foreign">cerche</span>, from <span class="foreign">cerchier</span> (see <a href="/etym/s/search" target="_blank">search</a> (v.)). <span class="foreign">Search warrant</span> attested from 1739.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

    var transjs = "/js/sttrans.js";

    window.yd_sttrans = function() {
        var cstt = '';
        if (window.localStorage) {
            cstt = window.localStorage.getItem('stt');
        }
        var handler = document.getElementById('sttrans');

        var _trans = function(is_trans) {
            var t = is_trans;
            if(typeof(trans) == 'undefined') {
                var jselm = document.createElement('script');
                jselm = document.body.appendChild(jselm);
                if(typeof(jselm) != 'undefined') {
                    jselm.onreadystatechange = function() {
                        if(this.readyState == 'loaded' || this.readyState == 'complete') {
                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

        if(cstt) {
            _trans(); if(handler) handler.innerHTML = '简体版';
        } else {
            var user_lang = window.navigator.language || window.navigator.browserLanguage || window.navigator.userLanguage;
            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            } else {
                _trans();handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        };
        handler.onclick = _evth;
    };
    yd_sttrans();

    function getMeaning() {
        var word = $('#word').val().replace(/\u3000/g, ' ').trim();
        $.getJSON(('https://fanyi.youdao.com/openapi.do?type=data&doctype=jsonp&version=1.1&keyfrom=neteaseopen&key=1532272597&callback=?&q=' + encodeURIComponent(word)), function(data) {
            if (data.basic) {
                $('#yd-word-meaning').html('<ul><li>' + data.basic.explains.join('</li><li>') + '</li></ul>');
                $('#yd-word-meaning li').html(function(index, text) {
                    return text.trim().replace(/^[a-z]+\./, '<strong>$&</strong>');
                });
            } else {
                var word_lang = 'en';
                if (word.search(/[\u4e00-\u9fa5]/) >= 0) {
                    word_lang = 'zh';
                }
                var str_query = encodeURIComponent(word) + '&lang=' + word_lang;
                $.getJSON(('/ciyuan/ajax-meanings.php?word=' + str_query), function(data) {
                    if (data.is_success) {
                        var is_not_have_info = true;
                        if (data.smart_dict) {
                            $('#yd-word-meaning').html('<ul style="margin-bottom:15px;"><li>' + data.smart_dict.join('</li><li>') + '</li></ul>');
                            $('#yd-word-meaning li').html(function(index, text) {
                                return text.trim().replace(/^[a-z]+\./, '<strong>$&</strong>');
                            });
                            is_not_have_info = false;
                        }

                        if (data.sug_words) {
                            var str_html = '<div class="panel panel-default"><div class="panel-body"><strong style="color:#666;">搜索建议：</strong>';
                            data.sug_words.forEach(function(word) {
                                str_html += '<a href="/w/' + word.replace(/ /g, '_') + '">' + word + '</a>, ';
                            });
                            str_html = str_html.slice(0, -2) + '</div></div>';
                            $('#yd-word-meaning').after(str_html);
                            is_not_have_info = false;
                        }

                        if (data.bing_trans) {
                            var yd_trans = data.bing_trans;
                            if (word_lang == 'en') {
                                if (yd_trans.search(/[\u4e00-\u9fa5]/) >= 0) {
                                    $('#yd-word-meaning').after('<div class="panel panel-default"><div class="panel-body"><strong style="color:#666;">翻译：</strong>' + yd_trans + '</div></div>');
                                    is_not_have_info = false;
                                }
                            } else {
                                if (yd_trans.search(/[\u4e00-\u9fa5]/) == -1) {
                                    $('#yd-word-meaning').after('<div class="panel panel-default"><div class="panel-body"><strong style="color:#666;">翻译：</strong>' + yd_trans + '</div></div>');
                                    is_not_have_info = false;
                                }
                            }
                        }

                        if (is_not_have_info) {
                            $('#yd-word-meaning').html('对不起，没有查询到该单词，建议使用百度搜索试试：<a target="_blank" href="https://www.baidu.com/s?ie=utf-8&wd=' + encodeURIComponent(word) + '">' + word + '</a>');
                        }
                    } else {
                        $('#yd-word-meaning').html('对不起，没有查询到该单词，建议使用百度搜索试试：<a target="_blank" href="https://www.baidu.com/s?ie=utf-8&wd=' + encodeURIComponent(word) + '">' + word + '</a>');
                    }
                });
            }
        });
    }
