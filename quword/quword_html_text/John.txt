                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/John">John</a> 约翰</span><p>常见人名，来自希伯来语Yohanan,来自yoh,上帝，缩写自Yahweh,hanan,亲切的，仁慈的，字面意思即上帝是仁慈的。</p><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/john">john</a> 厕所，茅房，嫖客</span><p>来自人名John,俚语用于指警察，嫖客，神父，祭司等各种职业，现主要用于指厕所，茅房。比较jack,jock,jenny.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/John">John</a></dt><dd>masc. proper name, mid-12c., from Medieval Latin <span class="foreign">Johannes</span>, from Late Latin <span class="foreign">Joannes</span>, from Greek <span class="foreign">Ioannes</span>, from Hebrew <span class="foreign">Yohanan</span> (longer form <span class="foreign">y'hohanan</span>) literally "Jehovah has favored," from <span class="foreign">hanan</span> "he was gracious."&#13; <br /><br />&#13; As the name of John the Baptist and John the Evangelist, it was one of the most common Christian given names, and in England by early 14c. it rivaled <span class="foreign">William</span> in popularity. The Old French form was <span class="foreign">Jean</span>, but in England its variants <span class="foreign">Johan</span>, <span class="foreign">Jehan</span> yielded <span class="foreign">Jan</span>, <span class="foreign">Jen</span> (also compare surname <span class="foreign">Jensen</span>). Welsh form was <span class="foreign">Ieuan</span> (see <a href="/etym/s/Evan" target="_blank">Evan</a>), but <span class="foreign">Ioan</span> was adopted for the Welsh Authorized Version of the Bible, hence frequency of <span class="foreign">Jones</span> as a Welsh surname.</dd><dt><a target="_blank" href="/etym/s/john">john (n.)</a></dt><dd>"toilet," 1932, probably from <a href="/etym/s/jakes" target="_blank">jakes</a>, used for "toilet" since 15c. Meaning "prostitute's customer" is from 1911, probably from the common, and thus anonymous, name by which they identified themselves. Meaning "policeman" is 1858, from shortening of <span class="foreign">johndarm</span>, jocular anglicization of <a href="/etym/s/gendarme" target="_blank">gendarme</a>.</dd></dl></div></div>

Sue and <b>John</b> were especially thrilled with this award. </dt> <dd>休和约翰对于获得这个奖项感到兴奋不已。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>4. At 84, <b>John</b> feels his age precludes too much travel. </dt> <dd>84岁的约翰感到自己年事已高，不能进行太多的旅行。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>5.     document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
