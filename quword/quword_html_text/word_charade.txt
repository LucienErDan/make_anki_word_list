                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/charade">charade</a> 伪装，猜字谜游戏</span><p>可能来自char，叽叽喳喳的声音，代指字谜游戏中的一些暗示。-ade, 名词后缀。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/charade">charade (n.)</a></dt><dd>1776, from French <span class="foreign">charade</span> (18c.), probably from Provençal <span class="foreign">charrado</span> "long talk, chatter," of obscure origin, perhaps from <span class="foreign">charrar</span> "to chatter, gossip," of echoic origin. Compare Italian <span class="foreign">ciarlare</span>, Spanish <span class="foreign">charlar</span> "to talk, prattle." Originally not silent, but relying rather on enigmatic descriptions of the words or syllables.&#13; <blockquote>&#13; As we have ever made it a Rule to shew our Attention to the Reader, by 'catching the Manners living, as they rise,' as Mr. Pope expresses it, we think ourselves obliged to give Place to the following Specimens of a new Kind of SMALL WIT, which, for some Weeks past, has been the Subject of Conversation in almost every Society, from the Court to the Cottage. The CHARADE is, in fact, a near Relation of the old Rebus. It is usually formed from a Word of two Syllables; the first Syllable is described by the Writer; then the second; they are afterwards united and the whole Word marked out .... [supplement to "The Universal Magazine of Knowledge and Pleasure," volumes 58-59, 1776]&#13; </blockquote>&#13; Among the examples given are:&#13; <br /><br />&#13; My first makes all nature appear of one face;&#13; <br />At the next we find music, and beauty and grace;&#13; <br />And, if this Charade is most easily read,&#13; <br />I think that the third shou'd be thrown at my head.&#13; <br /><br />[The answer is "snow-ball."]&#13; <br /><br />&#13; The silent form, the main modern form, was at first a variant known as <span class="foreign">dumb charades</span> and at first it was not a speed contest; rather it adhered to the old pattern, and the performing team acted out all the parts in order before the audience team began to guess.&#13; <blockquote>&#13; There is one species of charade which is performed solely by "dumb motions," somewhat resembling the child's game of "trades and professions"; but the acting charade is a much more amusing, and more difficult matter. ["Goldoni, and Modern Italian Comedy," in "The Foreign And Colonial Quarterly Review," Volume 6, 1846]&#13; </blockquote>&#13; An 1850 book, "Acting Charades," reports that <span class="foreign">Charades en Action</span> were all the rage in French society, and that "Lately, the game has been introduced into the drawing-rooms of a few mirth-loving Englishmen. Its success has been tremendous." Welsh <span class="foreign">siarad</span> obviously is a loan-word from French or English, but its meaning of "speak, a talk" is closer to the Provençal original.</dd></dl></div></div>
                                    <div class="page-header"><h3><a href="/liju/w/charade">权威例句</a></h3></div>
            <div class="row" id="yd-liju">
                 <dl><dt>1. I wondered why he had gone through the elaborate <b>charade</b>. </dt> <dd>我不明白，他为什么要大费周章，装腔作势。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. Their whole marriage had been a <b>charade</b>—they had never loved each other. </dt> <dd>他们的整个婚姻都是在做戏—他们从未相爱过。</dd> <p>来自《权威词典》</p> </dl><dl><dt>3. He wasn't really upset — his behaviour was just a <b>charade</b>. </dt> <dd>他并不是真正心绪烦乱——他的行为只是荒谬的伪装.</dd> <p>来自《简明英汉词典》</p> </dl><dl><dt>4. I refused to go along with their pathetic <b>charade</b>. </dt> <dd>我拒不跟他们摆出那副可悲的装模作样的姿态.</dd> <p>来自《简明英汉词典》</p> </dl><dl><dt>5. She struggled to maintain the <b>charade</b> of not being afraid. </dt> <dd>她竭力装出不害怕的样子.</dd> <p>来自《简明英汉词典》</p> </dl>                 <dl><p>[ <a href="/liju/w/charade">charade 造句</a> ]</p></dl>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }

            _trans(); if(handler) handler.innerHTML = '简体版';

            if ((user_lang == 'zh-TW') && (cstt !== '')) {

                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';
