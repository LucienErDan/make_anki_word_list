                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/might">might</a> 强大力量，威力，也许</span><p>来自古英语miht,来自Proto-Germanic*makhti,来自PIE*magh,使能够，赋予力量，词源同may,magic.引申词义强大力量，威力。后用于助动词表也许。比较may.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/might">might</a></dt><dd><b>might: </b><span>[OE] <i>Might </i>goes back ultimately to Indo- European *<i>mag</i>- ‘be able, have power’, the same base as produced the auxiliary verb <i>may</i>. The noun <i>might </i>was formed with the Germanic suffix *-<i>tiz</i>, which also gave German and Dutch <i>macht </i>‘power’; and the verb <i>might</i>, the past form of <i>may</i>, contains the past inflectional suffix (in modern English -(<i>e)d</i>).</span><br><span class="cognate">=&gt; <a href="/etym/s/may" target="_blank">may</a></span></dd><dt><a target="_blank" href="/etym/s/might">might (v.)</a></dt><dd>Old English <span class="foreign">mihte, meahte,</span> originally the past tense of <span class="foreign">may</span> (Old English <span class="foreign">magen</span> "to be able"), thus "*may-ed." See <a href="/etym/s/may" target="_blank">may</a> (v.). The first record of <span class="foreign">might-have-been</span> is from 1848.</dd><dt><a target="_blank" href="/etym/s/might">might (n.)</a></dt><dd>Old English <span class="foreign">miht</span>, earlier <span class="foreign">mæht</span> "might, bodily strength, power, authority, ability," from Proto-Germanic <span class="foreign">*makhti-</span> (cognates: Old Norse <span class="foreign">mattr</span>, Old Frisian, Middle Dutch, Dutch <span class="foreign">macht</span>, Old High German <span class="foreign">maht</span>, German <span class="foreign">Macht</span>, Gothic <span class="foreign">mahts</span>), Germanic suffixed form of PIE root <span class="foreign">*magh-</span> (1) "be able, have power" (see <a href="/etym/s/may" target="_blank">may</a> (v.)).</dd></dl></div></div>
                                    <div class="page-header"><h3><a href="/liju/w/might">权威例句</a></h3></div>
            <div class="row" id="yd-liju">
                 <dl><dt>1. I thought you <b>might</b> like to read the enclosed. </dt> <dd>我想你或许想要读一下信封里的内容。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. Today I <b>might</b> hesitate to turn in a burglar. </dt> <dd>现在让我把入室窃贼送交警方，我可能会有所顾虑。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>3. Things <b>might</b> have been different if I'd talked a bit more. </dt> <dd>如果当时我再多说一点的话，结果或许会不一样。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>4. I <b>might</b> buy one for a friend's birthday as a giggle. </dt> <dd>我可能会买一件送给朋友当生日礼物，逗他开心。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>5. I thought a phonetic spelling <b>might</b> aid in pronunciation. </dt> <dd>我想语音拼写可能有助于发音。</dd> <p>来自柯林斯例句</p> </dl>                 <dl><p>[ <a href="/liju/w/might">might 造句</a> ]</p></dl>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
