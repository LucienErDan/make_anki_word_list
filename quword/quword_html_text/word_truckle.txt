            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/truckle">truckle</a> 小轮，脚轮</span><p>来自希腊语 trokhos,轮子，词源同 truck.-le,小词后缀。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/truckle">truckle</a></dt><dd><b>truckle: </b><span>[15] A <i>truckle </i>is a ‘small wooden wheel or caster’. The word was originally used for a ‘pulley’ (an application which has now largely died out), and it was borrowed from Anglo- Norman <i>trocle</i>. This in turn came via Latin <i>trochlea </i>‘system of pulleys’ (source of English <i>trochlea </i>[17], an anatomical term for a ‘structure resembling a pulley’) from Greek <i>trokhilíā </i>‘pulley, system of pulleys, roller, etc’. <i>Trucklebed </i>was a term applied to a sort of low bed on casters that could be pushed under a larger bed when not in use, and the notion of sleeping in the truckle-bed, ‘beneath’ someone in the higher main bed, led in the 17th century to the use of <i>truckle </i>as a verb meaning ‘be subservient’.</span><br><span class="cognate">=&gt; <a href="/etym/s/trochlea" target="_blank">trochlea</a>, <a href="/etym/s/truck" target="_blank">truck</a></span></dd><dt><a target="_blank" href="/etym/s/truckle">truckle (n.)</a></dt><dd>"small wheel or roller," late 14c., from Anglo-French <span class="foreign">trocle</span>, from Latin <span class="foreign">trochlea</span> "a small wheel, sheaf of a pulley," from Greek <span class="foreign">trokhileia</span> "a system of pulleys," from <span class="foreign">trokhos</span> "wheel," from <span class="foreign">trekhein</span> "to run," from PIE root <span class="foreign">*dhregh-</span> "to run" (cognates: Old Irish <span class="foreign">droch</span> "wheel," Lithuanian <span class="foreign">pa-drosti</span> "to run fast"). <span class="foreign">Truckle bed</span> "small bed on wheels that can be stowed under a larger bed" is from mid-15c.</dd><dt><a target="_blank" href="/etym/s/truckle">truckle (v.)</a></dt><dd>"give up or submit tamely," 1610s, originally "sleep in a truckle bed" (see <a href="/etym/s/truckle" target="_blank">truckle</a> (n.)). Meaning "give precedence, assume a submissive position" (1650s, implied in <span class="foreign">truckling</span>) is perhaps in reference to that type of bed being used by servants and inferiors or simply occupying the lower position. Related: <span class="foreign">Truckled</span>; <span class="foreign">truckling</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
