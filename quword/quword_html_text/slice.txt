                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/slice">slice</a> 切，割，划，切成薄片</span><p>来自辅音丛 sl-,砍，劈，分开，比较 slab,slip,slat,slit,slot.引申诸相关词义。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/slice">slice</a></dt><dd><b>slice: </b><span>[14] <i>Slice </i>comes from Old French <i>esclice </i>‘splinter’, a derivative of the verb <i>esclicier </i>‘reduce to splinters, shatter’. This in turn was acquired from Frankish *<i>slītjan</i>, a descendant of prehistoric Germanic *<i>slītan </i>‘slit’ (source of English <i>slit </i>and possibly of <i>slat </i>and <i>slate</i>). English originally took over the word’s French meaning, but this had died out by the end of the 16th century. The modern sense ‘piece cut from something’ is first recorded in the early 15th century.</span><br><span class="cognate">=&gt; <a href="/etym/s/slit" target="_blank">slit</a></span></dd><dt><a target="_blank" href="/etym/s/slice">slice (n.)</a></dt><dd>c. 1300, "a fragment," from Old French <span class="foreign">escliz</span> "splinter, fragment" (Modern French <span class="foreign">éclisse</span>), a back-formation from <span class="foreign">esclicier</span> "to splinter, shatter, smash," from Frankish <span class="foreign">*slitan</span> "to split" or some other Germanic source (compare Old High German <span class="foreign">slihhan</span>; see <a href="/etym/s/slit" target="_blank">slit</a> (v.)). Meaning "piece cut from something" emerged early 15c. Meaning "a slicing stroke" (in golf, tennis) is recorded from 1886. <span class="foreign">Slice of life</span> (1895) translates French <span class="foreign">tranche de la vie</span>, a term from French Naturalist literature.</dd><dt><a target="_blank" href="/etym/s/slice">slice (v.)</a></dt><dd>late 15c., from Middle French <span class="foreign">esclicier</span>, from Old French <span class="foreign">escliz</span> (see <a href="/etym/s/slice" target="_blank">slice</a> (n.)). Golfing sense is from 1890. Related: <span class="foreign">Sliced</span>; <span class="foreign">slicing</span>. <span class="foreign">Sliced bread</span> introduced 1958; <span class="foreign">greatest thing since ...</span> first attested 1969.&#13; <blockquote>&#13; No matter how thick or how thin you slice it it's still baloney. [Carl Sandburg, "The People, Yes," 1936]&#13; </blockquote></dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                        if(this.readyState == 'loaded' || this.readyState == 'complete') {
                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

        if(cstt) {
            _trans(); if(handler) handler.innerHTML = '简体版';
        } else {

            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';

                    window.localStorage.setItem('stt', cstt);
