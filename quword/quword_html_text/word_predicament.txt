                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/predicament">predicament</a> 情况，困境，窘境</span><p>来自拉丁语praedicamentum,预测之事，分类，品质，词源同predicate.后借用该拉丁词来翻译希腊语kategoria,分类，该词为古希腊哲学家亚里士多德对当时的社会进行分类后创造出来的词，共十类，即being,quantity,quality,relation,place,time,posture,possession,action,passion.引申词义情况，状况，后特别用于指不好的情况，即困难，窘境。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/predicament">predicament</a></dt><dd><b>predicament: </b><span>[14] <i>Predicament </i>was originally a technical term in logic, denoting a ‘category of attributes which may be asserted of a thing’. It broadened out in the 16th century to ‘situation’, but it does not seem to have been until the 18th century that the specific modern sense ‘awkward situation’ became established. The word comes from late Latin <i>praedicāmentum</i>, a derivative of <i>praedicāre </i>‘proclaim’ (source of English <i>preach </i>and <i>predicate </i>[16]). This was a compound verb formed from the prefix <i>prae</i>- ‘in front of’, hence ‘in public’ and <i>dicāre </i>‘make known’.</span><br><span class="cognate">=&gt; <a href="/etym/s/preach" target="_blank">preach</a>, <a href="/etym/s/predicate" target="_blank">predicate</a></span></dd><dt><a target="_blank" href="/etym/s/predicament">predicament (n.)</a></dt><dd>early 15c., "category, class; one of Aristotle's 10 categories," from Medieval Latin <span class="foreign">predicamentum</span>, from Late Latin <span class="foreign">praedicamentum</span> "quality, category, something predicted, that which is asserted," from Latin <span class="foreign">praedicatus</span>, past participle of <span class="foreign">praedicare</span> (see <a href="/etym/s/predicate" target="_blank">predicate</a>). <span class="foreign">Praedicamentum</span> is a loan-translation of Greek <span class="foreign">kategoria</span>, Aristotle's word. The meaning "unpleasant situation" is first recorded 1580s.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
