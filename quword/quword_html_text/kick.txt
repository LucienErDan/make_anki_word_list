            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/kick">kick</a> 踢</span><p>词源不详，可能来自拟声词，踢东西的声音。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/kick">kick</a></dt><dd><b>kick: </b><span>[14] <i>Kick </i>is one of the mystery words of English. It first appears towards the end of the 14th century, but no one knows where it came from, and it has no relatives in the other Indo- European languages. It may have been a Scandinavian borrowing.</span><br></dd><dt><a target="_blank" href="/etym/s/kick">kick (v.)</a></dt><dd>late 14c., "to strike out with the foot" (earliest in biblical phrase now usually rendered as <span class="foreign">kick against the pricks</span>), of uncertain origin, perhaps from Old Norse <span class="foreign">kikna</span> "bend backwards, sink at the knees." "The doubts OED has about the Scandinavian origin of <span class="foreign">kick</span> are probably unfounded" [Liberman]. Related: <span class="foreign">Kicked</span>; <span class="foreign">kicking</span>.&#13; <br /><br />&#13; Figurative sense of "complain, protest, rebel against" (late 14c.) probably is from the Bible verse. Slang sense of "die" is attested from 1725 (<span class="foreign">kick the wind</span> was slang for "be hanged," 1590s; see also <a href="/etym/s/bucket" target="_blank">bucket</a>). Meaning "to end one's drug habit" is from 1936. <span class="foreign">Kick in</span> "contribute" is from 1908; <span class="foreign">kick out</span> "expel" is from 1690s. To <span class="foreign">kick oneself</span> in self-reproach is from 1891. The children's game of <span class="foreign">kick the can</span> is attested from 1891.</dd><dt><a target="_blank" href="/etym/s/kick">kick (n.)</a></dt><dd>1520s, from <a href="/etym/s/kick" target="_blank">kick</a> (v.). Meaning "recoil (of a gun) when fired" is from 1826. Meaning "surge or fit of pleasure" (often as <span class="foreign">kicks</span>) is from 1941; originally literally, "stimulation from liquor or drugs" (1844). <span class="foreign">The kick</span> "the fashion" is c. 1700.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
