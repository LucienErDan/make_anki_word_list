                                                                        <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/special">special</a></dt><dd><b>special: </b><span>[13] Latin <i>speciēs </i>originally denoted the ‘outward aspect’, the ‘look’ of something (it was derived from <i>specere </i>‘look’, source of English <i>spectacle</i>, <i>spectator</i>, <i>spy</i>, etc). It later evolved metaphorically to ‘type, kind’, and in that sense was adopted by English as <i>species </i>[16] (<i>spice </i>is ultimately the same word). From it was derived the adjective <i>speciālis </i>‘of a particular type’, which has given English <i>special </i>(<i>especial </i>[14] came via Old French <i>especial</i>). Other derivatives have given English <i>specific </i>[17], <i>specify </i>[13], <i>specimen </i>[17], and <i>specious </i>[17] (from Latin <i>speciōsus </i>‘good-looking’).</span><br><span class="cognate">=&gt; <a href="/etym/s/spectacle" target="_blank">spectacle</a>, <a href="/etym/s/spectator" target="_blank">spectator</a>, <a href="/etym/s/spice" target="_blank">spice</a>, <a href="/etym/s/spy" target="_blank">spy</a></span></dd><dt><a target="_blank" href="/etym/s/special">special (adj.)</a></dt><dd>c. 1200, "better than ordinary," from Old French <span class="foreign">special</span>, <span class="foreign">especial</span> "special, particular, unusual" (12c., Modern French <span class="foreign">spécial</span>) and directly from Latin <span class="foreign">specialis</span> "individual, particular" (source also of Spanish <span class="foreign">especial</span>, Italian <span class="foreign">speziale</span>), from <span class="foreign">species</span> "appearance, kind, sort" (see <a href="/etym/s/species" target="_blank">species</a>).&#13; <br /><br />&#13; Meaning "marked off from others by some distinguishing quality" is recorded from c. 1300; that of "limited as to function, operation, or purpose" is from 14c. <span class="foreign">Special effects</span> first attested 1951. <span class="foreign">Special interests</span> in U.S. political sense is from 1910. <span class="foreign">Special pleading</span> first recorded 1680s, a term that had a sound legal meaning once but now is used generally and imprecisely. <span class="foreign">Special education</span> in reference to those whose learning is impeded by some mental or physical handicap is from 1972.</dd><dt><a target="_blank" href="/etym/s/special">special (n.)</a></dt><dd>"sweetheart, lover; special person or thing," c. 1300, from <a href="/etym/s/special" target="_blank">special</a> (adj.) or from noun use of the adjective in Old French. Meaning "special train" is attested from 1866.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

        if(cstt) {
            _trans(); if(handler) handler.innerHTML = '简体版';
        } else {

            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';

                    window.localStorage.setItem('stt', cstt);
