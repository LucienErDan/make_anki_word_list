                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/than">than</a> 比，比较</span><p>来自 then 的拼写变体，并于 18 世纪开始成为两个不同的词，并产生词义上的分化。如 A is bigger than B,原义为 A is bigger,then B.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/than">than</a></dt><dd><b>than: </b><span>[OE] <i>Than </i>is ultimately the same word as <i>then</i>, and the two were used interchangeably until the end of the 17th century. It is not clear how the adverb came to be used as a conjunction denoting comparison, although it is possible that the comparison implicit in expressions like ‘This one is better; then there is that one’ may have led on to ‘This one is better than that one’.</span><br><span class="cognate">=&gt; <a href="/etym/s/then" target="_blank">then</a></span></dd><dt><a target="_blank" href="/etym/s/than">than (conj.)</a></dt><dd>Old English <span class="foreign">þan</span>, conjunctive particle used after a comparative adjective or adverb, from <span class="foreign">þanne</span>, <span class="foreign">þænne</span>, <span class="foreign">þonne</span> "then" (see <a href="/etym/s/then" target="_blank">then</a>). Developed from the adverb <span class="foreign">then</span>, and not distinguished from it by spelling until c. 1700.&#13; <br /><br />&#13; The earliest use is in West Germanic comparative forms introducing the second member, i.e. <span class="foreign">bigger than</span> (compare Dutch <span class="foreign">dan</span>, German <span class="foreign">denn</span>), which suggests a semantic development from the demonstrative sense of <span class="foreign">then</span>: <span class="foreign">A is bigger than B</span>, evolving from <span class="foreign">A is bigger, then ("after that") B</span>. Or the word may trace to Old English <span class="foreign">þonne</span> "when, when as," such as <span class="foreign">"When as" B is big, A is more (so)</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
