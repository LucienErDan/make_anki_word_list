                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/phlegm">phlegm</a> 痰，冷淡，迟钝，冷静，镇定</span><p>来自拉丁语phlegma,痰，黏液（古医学体液理论构成人体的血液，黏液，胆汁，黑胆汁四种体液之一），来自希腊语phlegma,发热，由体热所致的多痰症，来自phlegein,燃烧，发热，来自PIE*bhel,照耀，燃烧，词源同blaze,bleach.后在中世纪医学理论认为体内多痰会致人冷淡，迟钝，但是，被误解为冷淡是冷静，镇定的表现，因此，产生模糊和矛盾的词义。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/phlegm">phlegm</a></dt><dd><b>phlegm: </b><span>[14] Greek <i>phlégma </i>denoted ‘bodily fluid produced by inflammation’ (it was a derivative of <i>phlégein </i>‘burn’, which went back to the same Indo-European base as produced English <i>flagrant</i>, <i>flame</i>, <i>fulminate</i>, and <i>phlox </i>[18] – in Greek literally ‘flame’). As Latin <i>phlegma </i>it came to be used for ‘body fluid’ in general, and was incorporated into the medieval system of bodily humours as a term for the ‘cold moist humour’, which induced sluggishness (whence the meaning of the derivative <i>phlegmatic </i>[16]).<br><br>This came to be associated in the late Middle Ages with ‘mucus, particularly as produced in the respiratory passage’. English acquired the word via Old French <i>fleume </i>as <i>fleume</i>, and did not revert to the latinate form until the 16th century.</span><br><span class="cognate">=&gt; <a href="/etym/s/flagrant" target="_blank">flagrant</a>, <a href="/etym/s/flame" target="_blank">flame</a>, <a href="/etym/s/fulminate" target="_blank">fulminate</a>, <a href="/etym/s/phlox" target="_blank">phlox</a></span></dd><dt><a target="_blank" href="/etym/s/phlegm">phlegm (n.)</a></dt><dd>late 14c., <span class="foreign">fleem</span> "viscid mucus" (the stuff itself and also regarded as a bodily humor), from Old French <span class="foreign">fleume</span> (13c., Modern French <span class="foreign">flegme</span>), from Late Latin <span class="foreign">phlegma</span>, one of the four humors of the body, from Greek <span class="foreign">phlegma</span> "humor caused by heat," lit "inflammation, heat," from <span class="foreign">phlegein</span> "to burn," related to <span class="foreign">phlox</span> (genitive <span class="foreign">phlogos</span>) "flame, blaze," from PIE <span class="foreign">*bhleg-</span> "to shine, flash," from root <span class="foreign">*bhel-</span> (1) "to shine, flash, burn" (see <a href="/etym/s/bleach" target="_blank">bleach</a> (v.)). Modern form is attested from c. 1660. The "cold, moist" humor of the body, in medieval physiology, it was believed to cause apathy.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }

                        if(this.readyState == 'loaded' || this.readyState == 'complete') {
                            try {trans(t)} catch(e){}
                        }
                    };
                    jselm.onload = function() {trans(t);};
                }
                jselm.src = transjs;
            } else { trans(t); }
        };

        if(cstt) {
            _trans(); if(handler) handler.innerHTML = '简体版';
        } else {

            if ((user_lang == 'zh-TW') && (cstt !== '')) {
                _trans(); if(handler) handler.innerHTML = '简体版';
                cstt = '1';
                if (window.localStorage) {
                    window.localStorage.setItem('stt', cstt);
                }
            }
        }
        if(!handler) return;

        var _evth = function () {
            if(cstt) {
                _trans(true);handler.innerHTML = '繁體版';
                cstt = '';

                    window.localStorage.setItem('stt', cstt);
