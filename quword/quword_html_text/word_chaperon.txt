                    <div class="row"><h3 id="yd-word">chaperon</h3></div>
                                        <div class="row" id="yd-word-pron">['ʃæpərɔn]</div>
                                        <div class="row" id="yd-word-meaning"><ul>
<li>vt. 陪伴；护送</li>
<li>n. 年长女伴（陪伴未婚少女上社交场所的）；监护人（青年社交聚会时在场的）</li>
<li>vi. 当女伴；当监护人</li>
<li>n. (Chaperon)人名；(西)查佩龙；(法)沙普龙</li>
</ul></div>

            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/chaperon">chaperon</a> 年长女伴</span><p>词源同cap, 帽子。-on, 大词后缀。喻指保护者。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/chaperon">chaperon</a></dt><dd><b>chaperon: </b><span>[14] A <i>chaperon </i>was originally a ‘hood’. The word comes from Old French <i>chaperon</i>, a derivative of <i>chape</i>, whose variant <i>cape </i>was the source of English <i>cape</i>, and goes back ultimately to late Latin <i>cappa </i>‘hood, cloak’. The word’s modern sense, ‘companion safeguarding propriety’, which first appears in English in the 18th century, arose from the general notion of a ‘hood’ as something that gives protection.</span><br><span class="cognate">=&gt; <a href="/etym/s/cape" target="_blank">cape</a></span></dd><dt><a target="_blank" href="/etym/s/chaperon">chaperon (n.)</a></dt><dd>1720, "woman accompanying a younger, unmarried lady in public," from French <span class="foreign">chaperon</span> "protector," especially "female companion to a young woman," earlier "head covering, hood" (c. 1400), from Old French <span class="foreign">chaperon</span> "hood, cowl" (12c.), diminutive of <span class="foreign">chape</span> "cape" (see <a href="/etym/s/cap" target="_blank">cap</a> (n.)). "... English writers often erroneously spell it <span class="foreign">chaperone</span>, app. under the supposition that it requires a fem. termination" [OED]. The notion is of "covering" the socially vulnerable one.<blockquote>&#13; "May I ask what is a chaperon?"&#13; <br />&#13; "A married lady; without whom no unmarried one can be seen in public. If the damsel be five and forty, she cannot appear without the matron; and if the matron be fifteen, it will do."&#13; <br />&#13; [Catharine Hutton, "The Welsh Mountaineer," London, 1817]&#13; </blockquote>&#13; The word had been used in Middle English in the literal sense "hooded cloak."</dd><dt><a target="_blank" href="/etym/s/chaperon">chaperon (v.)</a></dt><dd>"act as a chaperon," 1792, also <span class="foreign">chaperone</span>, from <a href="/etym/s/chaperon" target="_blank">chaperon</a> (n.), or from French <span class="foreign">chaperonner</span>, from <span class="foreign">chaperon</span> (n.). Related: <span class="foreign">Chaperoned</span>; <span class="foreign">chaperoning</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
