                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/magnesia">magnesia</a> 氧化镁，镁砂</span><p>来自拉丁语magnesia,来自希腊语Magnesia lithos,即来自Magnesia的石头，lithos,石头，词源同lithology.Magnesia为古希腊地名，来自当地人的族名Magnetes.因石头中发现镁砂而得名。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/magnesia">magnesia (n.)</a></dt><dd>late 14c., in alchemy, "main ingredient of the philosopher's stone," from Medieval Latin <span class="foreign">magnesia</span>, from Greek <span class="foreign">(he) Magnesia (lithos)</span> "the lodestone," literally "(the) Magnesian (stone)," from <span class="foreign">Magnesia</span>, region in Thessaly, which is said to be named for the native people name <span class="foreign">Magnetes</span>, which is of unknown origin. The ancient word, in this sense, has evolved into <a href="/etym/s/magnet" target="_blank">magnet</a>. But in ancient times the same word, <span class="foreign">magnes</span>, was used of lodestone as well as of a mineral commonly used in bleaching glass (modern <a href="/etym/s/pyrolusite" target="_blank">pyrolusite</a>, or manganese dioxide).&#13; <br /><br />&#13; In Middle Ages there was some attempt to distinguish lodestone as <span class="foreign">magnes</span> (masc.) and pyrolusite as <span class="foreign"> magnesia</span> (fem.). Meanwhile, in 18c., a white powder (magnesium carbonate) used as a cosmetic and toothpaste was sold in Rome as <span class="foreign">magnesia alba</span> ("white magnesia"). It was from this, in 1808, that Davy isolated <a href="/etym/s/magnesium" target="_blank">magnesium</a>. He wanted to call it <span class="foreign">magnium</span>, to stay as far as possible from the confused word <span class="foreign">magnesia</span>, but the name was adopted in the form <span class="foreign">magnesium</span>. Meanwhile from 16c. the other name of pyrolusite had been corrupted to <span class="foreign">manganese</span>, and when, in 1774, a new element was isolated from it, it came to be called <a href="/etym/s/manganese" target="_blank">manganese</a>.&#13; <br /><br /><span class="foreign">Magnesia</span> in its main modern sense of "magnesium oxide" (1755) is perhaps an independent formation from Latin <span class="foreign">magnes carneus</span> "flesh-magnet" (c. 1550), so called because it adheres strongly to the lips.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
