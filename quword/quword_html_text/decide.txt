                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/decide">decide</a> 决定</span><p>de-, 向下。-cid, 砍，击，词源同concise, incision. 引申义决定。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/decide">decide</a></dt><dd><b>decide: </b><span>[14] Etymologically, <i>decide </i>denotes a resolving of alternatives or difficulties by cutting through them as if with a knife or a sword – dealing with them ‘at a stroke’. The word comes, perhaps via French <i>décider</i>, from Latin <i>dēcidere</i>, a compound verb formed from the prefix <i>dē</i>- ‘off’ and <i>caedere </i>‘cut, strike’. It is not clear where this comes from, although Sanskrit <i>khid</i>- ‘press, tear’ has been compared.<br><br>Its other descendants in English include <i>chisel</i>, <i>cement</i>, <i>concise</i>, and <i>scissors</i>. (Other verbs for ‘decide’ which contain the basic meaning element ‘cut through’ or ‘separate’ include Latin <i>dēcernere </i>and German <i>entscheiden</i>.)</span><br><span class="cognate">=&gt; <a href="/etym/s/cement" target="_blank">cement</a>, <a href="/etym/s/chisel" target="_blank">chisel</a>, <a href="/etym/s/concise" target="_blank">concise</a>, <a href="/etym/s/excise" target="_blank">excise</a>, <a href="/etym/s/incision" target="_blank">incision</a>, <a href="/etym/s/precise" target="_blank">precise</a>, <a href="/etym/s/scissors" target="_blank">scissors</a></span></dd><dt><a target="_blank" href="/etym/s/decide">decide (v.)</a></dt><dd>late 14c., "to settle a dispute," from Old French <span class="foreign">decider</span>, from Latin <span class="foreign">decidere</span> "to decide, determine," literally "to cut off," from <span class="foreign">de-</span> "off" (see <a href="/etym/s/de-" target="_blank">de-</a>) + <span class="foreign">caedere</span> "to cut" (see <a href="/etym/s/-cide" target="_blank">-cide</a>). For Latin vowel change, see <a href="/etym/s/acquisition" target="_blank">acquisition</a>. Sense is of resolving difficulties "at a stroke." Meaning "to make up one's mind" is attested from 1830. Related: <a href="/etym/s/Decided" target="_blank">Decided</a>; <span class="foreign">deciding</span>.</dd></dl></div></div>

            <div class="row" id="yd-liju">
                 <dl><dt>1. A dentist may <b>decide</b> to extract the tooth to prevent recurrent trouble. </dt> <dd>牙医可能会决定拔掉那颗牙，以免反复发作。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. If the clubs cannot conclude a deal, an independent tribunal will <b>decide</b>. </dt> <dd>如果俱乐部之间无法达成协议，将由一个独立的仲裁委员会来裁定。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>4. The judges could not <b>decide</b> which category it belonged in. </dt> <dd>评委们无法判定它属于哪一类。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>5.     document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

                return String.fromCharCode(char_code);

        }
