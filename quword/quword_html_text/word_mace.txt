                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/Mace">Mace</a> 梅斯催泪气体</span><p>专有名词，来自mace,狼牙棒，武器。</p><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/mace">mace</a> 权杖，狼牙棒</span><p>来自古法语mace,短棍，节杖，来自拉丁语mateola,锤子，铁锤，词源同machete.后也用于指权杖，狼牙棒。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/mace">mace (n.1)</a></dt><dd>"heavy metal weapon, often with a spiked head," late 13c., from Old French <span class="foreign">mace</span> "a club, scepter" (Modern French <span class="foreign">masse</span>), from Vulgar Latin <span class="foreign">*mattea</span> (source also of Italian <span class="foreign">mazza</span>, Spanish <span class="foreign">maza</span> "mace"), from Latin <span class="foreign">mateola</span> (in Late Latin also <span class="foreign">matteola</span>) "a kind of mallet." The Latin word perhaps is cognate with Sanskrit <span class="foreign">matyam</span> "harrow, club," Old Church Slavonic <span class="foreign">motyka</span> "mattock," Old High German <span class="foreign">medela</span> "plow" [Klein]. As a symbol of authority or office from mid-15c.</dd><dt><a target="_blank" href="/etym/s/mace">mace (n.2)</a></dt><dd>"spice made from dry outer husk of nutmeg," late 14c., from Old French <span class="foreign">macis</span> (in English taken as a plural and stripped of its <span class="foreign">-s</span>), of uncertain origin, sometimes said to be a scribal error for Latin <span class="foreign">macir</span>, the name of a red spicy bark from India, but OED finds this etymology unlikely.</dd><dt><a target="_blank" href="/etym/s/Mace">Mace (n.3)</a></dt><dd>chemical spray originally used in riot control, 1966, technically <span class="foreign">Chemical Mace</span>, a proprietary name (General Ordnance Equipment Corp, Pittsburgh, Pa.), probably so called for its use as a weapon, in reference to <a href="/etym/s/mace" target="_blank">mace</a> (1). The verb is first attested 1968. Related: <span class="foreign">Maced</span>; <span class="foreign">macing</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
