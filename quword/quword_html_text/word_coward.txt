                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/coward">coward</a> 胆小鬼</span><p>来自拉丁词cauda, 尾巴，词源同coda, queue. 原指动物夹着尾巴逃走，后指人胆小。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/coward">coward</a></dt><dd><b>coward: </b><span>[13] Etymologically, a <i>coward </i>seems to be ‘someone who runs away with his tail between his legs’. It comes from Old French <i>cuard</i>, which was based on *<i>cōda</i>, the Vulgar Latin descendant of Latin <i>cauda </i>‘tail’. (The apparently similar <i>cower </i>[13] is no relation, coming from Middle Low German *<i>kūren </i>‘lie in wait’.)</span><br></dd><dt><a target="_blank" href="/etym/s/coward">coward (n.)</a></dt><dd>mid-13c., from Old French <span class="foreign">coart</span> "coward" (no longer the usual word in French, which has now in this sense <span class="foreign">poltron</span>, from Italian, and <span class="foreign">lâche</span>), from <span class="foreign">coe</span> "tail," from Latin <span class="foreign">coda</span>, popular dialect variant of <span class="foreign">cauda</span> "tail," which is of uncertain origin + <span class="foreign">-ard</span>, an agent noun suffix denoting one that carries on some action or possesses some quality, with derogatory connotation (see <a href="/etym/s/-ard" target="_blank">-ard</a>).&#13; <br /><br />&#13; The word probably reflects an animal metaphoric sense still found in expressions like <span class="foreign">turning tail</span> and <span class="foreign">tail between legs</span>. <span class="foreign">Coart</span> was the name of the hare in Old French versions of "Reynard the Fox." Italian <span class="foreign">codardo</span>, Spanish <span class="foreign">cobarde</span> are from French.&#13; <blockquote>&#13; The identification of coward &amp; bully has gone so far in the popular consciousness that persons &amp; acts in which no trace of fear is to be found are often called <span class="foreign">coward(ly)</span> merely because advantage has been taken of superior strength or position .... [Fowler]&#13; </blockquote>&#13; As a surname (attested from 1255) it represents Old English <span class="foreign">cuhyrde</span> "cow-herd." Farmer has <span class="foreign">coward's castle</span> "a pulpit," "Because a clergyman may deliver himself therefrom without fear of contradiction or argument."</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

                return String.fromCharCode(char_code);

        }
