                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/gossamer">gossamer</a> 蛛丝，薄纱</span><p>gos, 鹅。-samer, 来自summer的变体，指深秋的短暂的阳气回升时节，小阳春。因这个时候常会出现形似鹅毛的蛛丝，或因这个时节为吃鹅正当令而得名。后指一种丝织品。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/gossamer">gossamer</a></dt><dd><b>gossamer: </b><span>[14] It would be pleasant to think that <i>gossamer</i>, originally ‘fine cobwebs’, is a descendant of an earlier <i>goose-summer</i>, but unfortunately there is not enough evidence to make this more than a conjecture. The theory goes as follows: mid-autumn is a time when geese for the table are plentiful (November was once known as <i>gänsemonat </i>‘geese-month’ in German), and so a warm period around then might have been termed <i>goose-summer </i>(we now call it an <i>Indian summer</i>); the silken filaments of <i>gossamer </i>are most commonly observed floating in the air on such warm autumnal days; and so the spiders’ webs were christened with the name of the season.</span><br></dd><dt><a target="_blank" href="/etym/s/gossamer">gossamer (n.)</a></dt><dd>c. 1300, "filmy substance (actually spider threads) found in fields of stubble in late fall," apparently from <span class="foreign">gos</span> "goose" (see <a href="/etym/s/goose" target="_blank">goose</a> (n.)) + <span class="foreign">sumer</span> "summer" (see <a href="/etym/s/summer" target="_blank">summer</a> (n.)). Not found in Old English. The reference might be to a fancied resemblance of the silk to goose down, or more likely it is shifted from an original sense of "late fall; Indian summer" because geese are in season then. Compare Swedish equivalent <span class="foreign">sommartrad</span> "summer thread," Dutch <span class="foreign">zommerdraden</span> (plural). The German equivalent <span class="foreign">mädchensommer</span> (literally "girls' summer") also has a sense of "Indian summer," and there was a Scottish <span class="foreign">go-summer</span> "period of summer-like weather in late autumn" (1640s, folk-etymologized as if from <span class="foreign">go</span>). Thus the English word originally might have referred to a warm spell in autumn before being transferred to a phenomenon especially noticeable then. Compare obsolete Scottish <span class="foreign">go-summer</span> "period of summer-like weather in late autumn." Meaning "anything light or flimsy" is from c. 1400; as a type of gauze used for veils, 1837. The adjective sense "filmy, light as gossamer" is attested from 1802.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
