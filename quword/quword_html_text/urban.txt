                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/urban">urban</a></dt><dd><b>urban: </b><span>[17] <i>Urban </i>comes from Latin <i>urbānus</i>, a derivative of <i>urbs </i>‘city’ (a word of unknown origin). It was preceded into English by <i>urbane </i>[16], which is essentially the same word, but came via Old French <i>urbaine</i>. It was originally used as <i>urban </i>is now, but after <i>urban </i>arrived it gradually took the metaphorical path to ‘smooth, sophisticated’. The derivatives <i>suburb </i>and <i>suburban </i>date from the 14th and 17th centuries respectively; and <i>suburbia </i>was coined in the 1890s.</span><br><span class="cognate">=&gt; <a href="/etym/s/suburban" target="_blank">suburban</a>, <a href="/etym/s/urbane" target="_blank">urbane</a></span></dd><dt><a target="_blank" href="/etym/s/urban">urban (adj.)</a></dt><dd>"characteristic of city life, pertaining to cities or towns," 1610s (but rare before 1830s), from Latin <span class="foreign">urbanus</span> "of or pertaining to a city or city life; in Rome," also "in city fashion, polished, refined, cultivated, courteous," but also sometimes "witty, facetious, bold, impudent;" as a noun, "city dweller," from <span class="foreign">urbs</span> (genitive <span class="foreign">urbis</span>) "city, walled town," of unknown origin.&#13; <br /><br />&#13; The word gradually emerged in this sense as <a href="/etym/s/urbane" target="_blank">urbane</a> became restricted to manners and styles of expression. In late 20c. American English gradually acquiring a suggestion of "African-American." <span class="foreign">Urban renewal</span>, euphemistic for "slum clearance," is attested from 1955, American English. <span class="foreign">Urban sprawl</span> recorded by 1958. <span class="foreign">Urban legend</span> attested by 1980.</dd><dt><a target="_blank" href="/etym/s/Urban">Urban</a></dt><dd>masc. proper name, from Latin <span class="foreign">urbanus</span> "refined, courteous," literally "of a city" (see <a href="/etym/s/urban" target="_blank">urban</a>).</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
