                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/uproar">uproar</a> 吵闹，骚乱</span><p>来自荷兰语uproer, 骚乱，up,向上，-roer, 摇动，搅拌，词源同rare, 翻炒，半熟。引申词义骚乱。而吵闹义可能是受roar影响的拼写讹误。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/uproar">uproar</a></dt><dd><b>uproar: </b><span>[16] <i>Uproar </i>has no direct etymological connection with <i>roar</i>. It originally meant ‘uprising, insurrection’, and was borrowed from Dutch <i>oproer</i>. This is a compound formed from <i>op </i>‘up’ and <i>roer </i>‘movement’. It was first used in English by William Tindale, in his 1526 translation of the Bible (for Acts 21:38 he has ‘that Egyptian which made an uproar, and led out into the wilderness about four thousand men’). The sense ‘loud outcry’, which was inspired of course by the similarity of <i>roar</i>, emerged as early as the 1540s.</span><br></dd><dt><a target="_blank" href="/etym/s/uproar">uproar (n.)</a></dt><dd>1520s, "outbreak of disorder, revolt, commotion," used by Tindale and later Coverdale as a loan-translation of German <span class="foreign">Aufruhr</span> or Dutch <span class="foreign">oproer</span> "tumult, riot," literally "a stirring up," in German and Dutch bibles (as in Acts xxi:38). From German <span class="foreign">auf</span> (Middle Dutch <span class="foreign">op</span>) "up" (see <a href="/etym/s/up" target="_blank">up</a> (adv.)) + <span class="foreign">ruhr</span> (Middle Dutch <span class="foreign">roer</span>) "a stirring, motion," related to Old English <span class="foreign">hreran</span> "to move, stir, shake" (see <a href="/etym/s/rare" target="_blank">rare</a> (adj.2)). Meaning "noisy shouting" is first recorded 1540s, probably by mistaken association with unrelated <span class="foreign">roar</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
