                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/a">a</a> 一 (个)</span><p>不定冠词an在发展过程中出现的缩略形式，应用于首字母为辅音的单词及h发音单词。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/a">a</a></dt><dd><b>a: </b><span>[OE] The indefinite article in English is ultimately identical with the word <i>one </i>(as is the case, even more obviously, in other European languages – French <i>un</i>, German <i>ein</i>, and so on). The ancestor of both <i>a(n) </i>and <i>one </i>was <i>ān</i>, with a long vowel, but in the Old English period it was chiefly used for the numeral; where we would use <i>a(n)</i>, the Anglo-Saxons tended not to use an article at all. <i>Ān </i>begins to emerge as the indefinite article in the middle of the 12th century, and it was not long before, in that relatively unemphatic linguistic environment, its vowel became weakened and shortened, giving <i>an</i>.<br><br>And at about the same time the distinction between <i>an </i>and <i>a </i>began to develop, although this was a slow process; until 1300 <i>an </i>was still often used before consonants, and right up to 1600 and beyond it was common before all words beginning with <i>h</i>, such as <i>house</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/one" target="_blank">one</a></span></dd><dt><a target="_blank" href="/etym/s/a">a (1)</a></dt><dd>indefinite article, mid-12c., a variation of Old English <span class="foreign">an</span> (see <a href="/etym/s/an" target="_blank">an</a>) in which the <span class="foreign">-n-</span> began to disappear before consonants, a process mostly complete by mid-14c. The <span class="foreign">-n-</span> also was retained before words beginning with a sounded <span class="foreign">-h-</span> until c. 1600; it still is retained by many writers before unaccented syllables in <span class="foreign">h-</span> or <span class="foreign">(e)u-</span>, but is now no longer normally spoken as such. The <span class="foreign">-n-</span> also lingered (especially in southern England dialect) before <span class="foreign">-w-</span> and <span class="foreign">-y-</span> through 15c.</dd><dt><a target="_blank" href="/etym/s/a">a (2)</a></dt><dd>as in <span class="foreign">twice a day</span>, etc., from Old English <span class="foreign">an</span> "on," in this case "on each." The sense was extended from time to measure, price, place, etc. The habit of tacking <span class="foreign">a</span> onto a gerund (as in <span class="foreign">a-hunting we will go</span>) died out 18c.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
