                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/slate">slate</a> 板岩，石板，石板瓦，写字板，清单，候选人名单</span><p>来自古法语 esclate,阴性格于 esclat,薄片，碎片，木条，即现拼写 slat.比较 marquis,侯爵， marquise,女侯爵，侯爵夫人。后 slat 多用于指木制板，slate 多用于指石制板，并引申比喻义 写字板，清单，候选人名单。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/slate">slate (n.)</a></dt><dd>mid-14c., from Old French <span class="foreign">esclate</span>, fem. of <span class="foreign">esclat</span> "split piece, splinter" (Modern French <span class="foreign">éclat</span>; see <a href="/etym/s/slat" target="_blank">slat</a>), so called because the rock splits easily into thin plates. As an adjective, 1510s. As a color, first recorded 1813. Sense of "a writing tablet" (made of slate), first recorded late 14c., led to that of "list of preliminary candidates prepared by party managers," first recorded 1842, from notion of being easily altered or erased. <span class="foreign">Clean slate</span> (1856) is an image from customer accounts chalked up in a tavern.</dd><dt><a target="_blank" href="/etym/s/slate">slate (v.)</a></dt><dd>1520s, "to cover with slates" (earlier <span class="foreign">sclatten</span>, late 15c.), from <a href="/etym/s/slate" target="_blank">slate</a> (n.). Meaning "propose, schedule" is from 1883; earlier "to nominate" (1804); the notion is of writing on a slate board. Related: <span class="foreign">Slated</span>; <span class="foreign">slating</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
