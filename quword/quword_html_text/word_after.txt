                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/after">after</a> 在后</span><p>来自PIE *apo, 从，后面，词源同前缀ab-, 单词of。-ter比较级后缀。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/after">after</a></dt><dd><b>after: </b><span>[OE] In the first millennium AD many Germanic languages had forms cognate with Old English <i>æfter </i>(Gothic <i>aftra</i>, for example, and Old Norse <i>aptr</i>), but, with the exception of Dutch <i>achter</i>, none survive. It is not clear what their ultimate origin is, but the suffix they share may well be a comparative one, and it is possible that they derive from a Germanic base *<i>af</i>- (represented in Old English <i>æftan </i>‘from behind’).<br><br>It has been suggested that this goes back to Indo-European *<i>ap</i>- (source of Latin <i>ab </i>‘away, from’ and English <i>of(f)</i>), in which case <i>after </i>would mean literally ‘more off’ – that is, ‘further away’. Nautical <i>aft </i>is probably a shortening of <i>abaft</i>, formed, with the prefixes <i>a</i>- ‘on’ and <i>be</i>- ‘by’, from Old English <i>æftan</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/of" target="_blank">of</a>, <a href="/etym/s/off" target="_blank">off</a></span></dd><dt><a target="_blank" href="/etym/s/after">after (prep.)</a></dt><dd>Old English <span class="foreign">æfter</span> "after, next, throughout, following in time, later," from Old English <span class="foreign">of</span> "off" (see <a href="/etym/s/of" target="_blank">of</a>) + <span class="foreign">-ter</span>, a comparative suffix; thus the original meaning was "more away, farther off." Compare Old Norse <span class="foreign">eptir</span> "after," Old High German <span class="foreign">aftar</span>, Gothic <span class="foreign">aftra</span> "behind." Cognate with Greek <span class="foreign">apotero</span> "farther off."&#13; <br /><br /><span class="foreign">After hours</span> "after regular working hours" is from 1861. <span class="foreign">Afterwit</span> "wisdom that comes too late" is attested from c. 1500 but seems to have fallen from use, despite being more needed now than ever. <span class="foreign">After you</span> as an expression in yielding precedence is recorded by 1650.</dd></dl></div></div>

            <div class="row" id="yd-liju">

    document.getElementById('yd-search').onsubmit = function() {

        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;

            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
