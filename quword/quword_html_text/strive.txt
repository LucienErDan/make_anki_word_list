                        <div class="page-header"><h3>记忆方法</h3></div>
            <div class="row" style="font-family:SimSun,serif;">1. strive, strife 是同源词(v <-->f)，因此它们的含义很相近；其实，stride 与 strive, strife 也是同源词，只是 stride 的引申义引申得比较远，因此仅从含义上看不出它们的同源性。<br />
2. 古高地德语：*strit- "fight, combat, contention; quarrel, dispute" => 古法语：*strit- / *strif- "quarrel, dispute" => strife => strive.<br />
3. 古高地德语：*strit- "fight, combat, contention" => *strid- => stride (其字面含义，其实就是两条腿之间的对峙、竞争，两条腿怎样才能形成对峙呢，当然就是叉开腿，形成跨步、跨立姿势)，由此引申出了如今的含义，straddle 就是 stride 的同源同引申义的词.<br />
4. *strid- => *strad- + -le (双写d加-le, 反复体后缀)。<br />
5. strive 和 strife 如今的含义出现了一定的分化，strive 多用来指褒义的努力、尽力在竞争、争斗中获得胜利，而 strife 则多来指中性的竞争、争斗的这种行为。</div>

                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/strive">strive</a> 努力，奋斗</span><p>来自中古英语 striven,艰难尝试，努力，奋斗，来自古法语动词 estriver,争吵，争斗，来自古 法语名词 estrif,争吵，争斗，改写自 estrit,争吵，来自 Proto-Germanic*strido,纷争，斗争，战 斗，词源同 stride.后主要用于褒义词义努力，奋斗，原词义见名词 strife,纷争，冲突。</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/strive">strive</a></dt><dd><b>strive: </b><span>[13] <i>Strive </i>was borrowed from Old French <i>estriver </i>‘quarrel, strive’. It is not certain where this came from, although it has been suggested that it was acquired from Old High German <i>strīt </i>‘contention’, a relative of English <i>stride</i>. <i>Strife </i>[13] comes from the associated Old French noun <i>estrif</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/strife" target="_blank">strife</a></span></dd><dt><a target="_blank" href="/etym/s/strive">strive (v.)</a></dt><dd>c. 1200, "quarrel, contend," from Old French <span class="foreign">estriver</span> "to quarrel, dispute, resist, struggle, put up a fight, compete," from <span class="foreign">estrif</span>, <span class="foreign">estrit</span> "quarrel" (see <a href="/etym/s/strife" target="_blank">strife</a>). It became a strong verb (past tense <span class="foreign">strove</span>) by rhyming association with <span class="foreign">drive</span>, <span class="foreign">dive</span>, etc. Meaning "try hard" is from early 14c. Related: <span class="foreign">Striving</span>.</dd></dl></div></div>
