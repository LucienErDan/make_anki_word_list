                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/tangible">tangible</a> 可接触的，可感知的，有形的，实际的</span><p>来自拉丁语 tangere,接触，触摸，来自 PIE*tag,接触，触摸，安排，词源同 integer,contact.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/tangible">tangible</a></dt><dd><b>tangible: </b><span>[16] <i>Tangible </i>means literally ‘touchable’. It comes via French <i>tangible </i>from late Latin <i>tangibilis</i>, a derivative of Latin <i>tangere </i>‘touch’. Other English words from this source include <i>tangent </i>[16], etymologically a line ‘touching’ a circle. Its past participle <i>tactus </i>has contributed <i>contact</i>, <i>intact</i>, and <i>tact</i>, while the base from which it was formed, *<i>tag</i>-, has also produced <i>contagion</i>, <i>contaminate</i>, <i>entire</i>, and <i>integrity</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/contact" target="_blank">contact</a>, <a href="/etym/s/contagion" target="_blank">contagion</a>, <a href="/etym/s/contaminate" target="_blank">contaminate</a>, <a href="/etym/s/intact" target="_blank">intact</a>, <a href="/etym/s/integrity" target="_blank">integrity</a>, <a href="/etym/s/tact" target="_blank">tact</a>, <a href="/etym/s/task" target="_blank">task</a>, <a href="/etym/s/taste" target="_blank">taste</a>, <a href="/etym/s/tax" target="_blank">tax</a></span></dd><dt><a target="_blank" href="/etym/s/tangible">tangible (adj.)</a></dt><dd>1580s, "capable of being touched," from Middle French <span class="foreign">tangible</span> and directly from Late Latin <span class="foreign">tangibilis</span> "that may be touched," from Latin <span class="foreign">tangere</span> "to touch" (see <a href="/etym/s/tangent" target="_blank">tangent</a> (adj.)). Sense of "material" (as in <span class="foreign">tangible reward</span>) is first recorded 1610s; that of "able to be realized or dealt with" is from 1709. Related: <span class="foreign">Tangibly</span>.</dd></dl></div></div>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
