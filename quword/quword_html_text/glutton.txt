                                                <div class="page-header"><h3>中文词源</h3></div>
            <div class="row" id="yd-ciyuan" style="font-family:SimSun,serif;"><span class="ciyuan-title"><a target="_blank" href="/ciyuan/s/glutton">glutton</a> 暴食，饕餮</span><p>来自PIE*gwele, 喉咙，吞食，词源同gullet.</p></div>
                                                <div class="page-header"><h3>英语词源</h3></div>
            <div class="row"><div class="thumbnail" id="yd-etym"><dl><dt><a target="_blank" href="/etym/s/glutton">glutton</a></dt><dd><b>glutton: </b><span>[13] Indo-European *<i>gel</i>- produced a wide range of descendants in the general semantic area ‘swallow’, among them Latin <i>gula </i>‘throat’ and its offspring French <i>gueule </i>‘mouth’ and English <i>gullet</i>; German <i>kehle </i>‘throat’; and Latin <i>gluttīre </i>‘swallow’, which was probably the ultimate source of English <i>glut </i>[14]. Another was Latin <i>gluttō </i>‘overeater’, which English acquired via Old French <i>gluton</i>.</span><br><span class="cognate">=&gt; <a href="/etym/s/glut" target="_blank">glut</a>, <a href="/etym/s/gullet" target="_blank">gullet</a></span></dd><dt><a target="_blank" href="/etym/s/glutton">glutton (n.)</a></dt><dd>"one who eats and drinks to excess," early 13c., from Old French <span class="foreign">gloton</span> "glutton;" also "scoundrel," a general term of abuse (Modern French <span class="foreign">glouton</span>), from Latin <span class="foreign">gluttonem</span> (nominative <span class="foreign">glutto</span>) "overeater," formed from <span class="foreign">gluttire</span> "to swallow," from <span class="foreign">gula</span> "throat," from PIE <span class="foreign">*gwele-</span> (3) "to swallow" (see <a href="/etym/s/glut" target="_blank">glut</a> (v.)). General sense in reference to one who indulges in anything to excess is from 1704. <span class="foreign">Glutton for punishment</span> is from pugilism; the phrase is from 1854, but the idea is older:&#13; <blockquote>&#13; Thus, Theocritus, in his Milling-match, calls Amycus "a glutton," which is well known to be the classical phrase at Moulsey-Hurst, for one who, like Amycus, takes a deal of <span class="foreign">punishment</span> before he is <span class="foreign">satisfied</span>. [Tom Moore, "Tom Crib's Memorial to Congress," 1819]&#13; </blockquote></dd></dl></div></div>
                                    <div class="page-header"><h3><a href="/liju/w/glutton">权威例句</a></h3></div>
            <div class="row" id="yd-liju">
                 <dl><dt>1. Ivy must be a <b>glutton</b> for punishment. </dt> <dd>艾维一定是个不怕吃苦的家伙。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>2. He was a <b>glutton</b> for hard work. </dt> <dd>他是个吃苦耐劳的人。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>3. She's a <b>glutton</b> for work . She stays late every evening. </dt> <dd>她是个工作狂,每天都很晚才下班.</dd> <p>来自《简明英汉词典》</p> </dl><dl><dt>4. I can't control my eating. It's hard when people don't understand and call you a <b>glutton</b>. </dt> <dd>我无法克制自己的食欲，当人们不理解而叫我贪吃鬼的时候心里别提有多难受了。</dd> <p>来自柯林斯例句</p> </dl><dl><dt>5. You're going to drive all the way to London and back in a day? You're a <b>glutton</b> for punishment, aren't you ? </dt> <dd>你打算一路开车到伦敦,一天就跑个来回? 你真不怕吃苦, 不是 吗 ?</dd> <p>来自《简明英汉词典》</p> </dl>                 <dl><p>[ <a href="/liju/w/glutton">glutton 造句</a> ]</p></dl>

    document.getElementById('yd-search').onsubmit = function() {
        var word = this.word.value.replace(/\u3000/g, ' ').replace(/(^\s+|\s+$)/g, '');
        if (word == '') {
            return false;
        } else if (word.search(/^[\w \-]+$/) == 0) {
            top.location.href = '/w/' + encodeURIComponent(word.replace(/ /g, '_'));
            return false;
        } else {
            this.word.value = word.replace(/[\uff01-\uffe1]/g, function(match) {
                var char_code = match.charCodeAt(0) - 0xfee0;
                return String.fromCharCode(char_code);
            });
        }
    }
